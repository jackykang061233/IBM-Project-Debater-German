Obwohl im weiten Sinne des Begriffs Smartphones sowohl Smartphone-PCs als auch Microsoft Smartphone-Chips in diese Kategorie passen. Microsofts Verwendung des Begriffs Smartphone umfasst nur spezielle Hardware-Geräte, die sich von Mobiltelefonen unterscheiden. Solche Smartphones wurden ursprünglich ohne Touchscreens konzipiert, die mit nur einer Hand effizienter betrieben werden sollen und in der Regel niedrigere Bildschirmauflösung als Taschen-PCs hatten. Microsoft konzentrierte sich auf die Smartphone-Plattform, um ein Gerät zu schaffen, das gut funktioniert, sowie ein Telefon- und Datengerät stärker integriert. Marktanteil von Windows Mobile im Smartphone-Markt stieg von seiner Einführung an, während neue Geräte freigegeben wurden. Nach dem Spitzenstand im Jahr 2007 verzeichnete sie einen Rückgang der Jahre. In Q1 2003 war Windows Mobile das drittgrößte Betriebssystem auf dem Markt für intelligentes und tragbares Gerät, hinter Symbian und Palm OS. In Q1 2004 machten Windows Mobile 23% des weltweiten Smartphone-Vertriebs aus. Windows Mobile wurde 2005 ins Leben gerufen, um Symbian als führendes mobiles OS bis 2010 zu werden. In Q3 2004 überstieg Windows Mobile (CE) das Palm OS, um das größte Smartphone-Betriebssystem zu werden. Microsoft hatte in Q4 2005 2,2 Millionen US-Dollar verbracht, was im gleichen Quartal im folgenden Jahr auf 3,5 Millionen angestiegen ist. Windows Mobile verzeichnete im Laufe des Jahres ein Wachstum zwischen 2005 und 2006 von 38,8 %, das nach der „Türkein Windows Mobile, um ihre starke Marktposition zu festigen“. Konnektivität In den frühen Jahren von Windows Mobile-Geräten konnten sie von einem ferngesteuerten Computer unter Einsatz von ActiveSync verwaltet und verschlüsselt werden; eine von Microsoft entwickelte Datenschnittstelle-Technologie und ein Protokoll, das ursprünglich 1996 veröffentlicht wurde. Hierdurch konnten Server, die Microsoft Exchange Server oder andere Varianten von Dritten betreiben, als persönlicher Informationsmanager agieren und Informationen wie E-Mail, Kalender-Bestellungen, Kontakte oder Internet-Abonnement austauschen. Mit der Freisetzung von Windows Vista wurde ActiveSync durch Windows Mobile Geräte Center ersetzt. Gerätezentrum ist mit Vista und Windows 7 enthalten und bietet viele Front-Endverbesserungen, die es einem Heimnutzer ermöglichen, PIM-Informationen mit Microsoft Outlook 2003 und später Fotos von Windows Photo Galerie, Videos oder Musik von Windows Media Player und Favoriten mit Internet Explorer abzustimmen, ohne dass ein Server wieder beendet werden muss. Geräte in dieser Zeit enthalten auch einen Basisfahrer, der mit dem Mobile Geräte Center kompatibel ist, so dass ein Nutzer einen Computer verbinden kann, ohne dass eine Konfiguration erforderlich ist.
