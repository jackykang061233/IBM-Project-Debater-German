Opera Mobile Store ist ein plattformunabhängiger Browser-App-Store für Mobilfunkbetreiber und eine digitale Anwendungsverteilungsplattform, die von mehr als 400.000 Entwicklern in der ganzen Welt genutzt wird und von Opera Software ASA gehalten wird. Im März 2011 wurde das Opera Mobile Store nach dem Erwerb von Handster, einem mobilen App Store-Plattform-Unternehmen, auf einer neuen Plattform neu gestartet und von einem Dritten betrieben. Mehr als 7 500 verschiedene Geräte auf Android, Java, Android OS, Symbian, iOS und Windows Mobile können von den Nutzern heruntergeladen werden. Opera Mobile Mehr als 1 Million App-Downloads pro Tag und ist über ein visuelles Geschenk mit dem OMS-Logo auf Speed Dial oder Start Seite von Opera Mini- oder Opera-Mobile-Browsern, die weltweit über 260 Millionen Geräte laufen, zugänglich. Der Speicher ist auch von jedem anderen Browser, einschließlich Desktop-Versionen, und auf jedem Betriebssystem von http://apps.opera.com zugänglich. Opera Mobile Lagern Sie dann den Nutzern auf der Grundlage der von dem Mobilfunkbetreiber oder dem ISP bereitgestellten Standortinformationen an das länderspezifische Lager. Mehr als 86 % der im Opera Mobile Store vorgesehenen Anwendungen für Android-getriebe sind kostenlos, während andere Zahlungen verlangen. Durchschnittlich sind 70 % bzw. 30 % der Kosten für alle Plattformen. Diese Anwendungen sind in der Regel auf eine bestimmte Kategorie ausgerichtet, darunter Videospiele, Geschäftsanwendungen, soziale Medien Apps, E-Books und andere. Opera Software kündigte am 8. März 2011 den Start von Opera Mobile Store an, der von Dritten betrieben wird. September 2011 Opera Software erwarb App Store-Plattform-Unternehmen Handster, das zu diesem Zeitpunkt die führende unabhängige Software für den Android-Markt war. Nach dem Erwerb von Handster hat die Opera Software seit dem Start des Opera Mobile Store eine große Reform vorgenommen. Opera Software kündigte am 27. Februar 2012 im Mobile World Congress in Barcelona einen neugestalteten Opera Mobile Store mit verbesserten Vertriebs- und Monetisationskapazitäten für Entwickler und bessere Anpassung an die Märkte für Weißzeichen an. Im Juli 2013 können Entwickler die Apps nun sowohl auf den Pool Store als auch auf den Opera Mobile Store stellen, was auf Vereinbarungen zwischen den beiden Unternehmen zurückzuführen ist. November 2014 haben Microsoft und Opera Software eine Vereinbarung unterzeichnet, um Nokia Store mit dem Opera Mobile Store als Standard-App-Store für Nokia-Telefone, Symbian und Nokia X Smartphones zu ersetzen. In der ersten Hälfte des Jahres 2015 wird erwartet, dass die Umstellung der Kunden von Nokia Store auf Opera Mobile Store abgeschlossen wird, an der der Punkt Nokia Store geschlossen wird. Anzahl der heruntergeladenen Anwendungen, wie in der offiziellen Opera Software-Pressemitteilung angegeben, erreichten bis Februar 27, 2012 Opera Mobile Store mehr als 30 Millionen monatliche Besuche und mehr als 45 Millionen monatliche App-Downloads. Anzahl der Apps im Speicher hat sich im Laufe des Jahres 2013 verdreifacht, während die Zahl der monatlichen Besucher an der App-Plattform der Opera im Oktober 2013 auf 75 Millionen gestiegen ist. Am Ende von Q4 2013 gab Opera Software bekannt, dass es einen neuen Meilenstein erreicht hat – 105 Millionen monatliche Besucher. Dies ist ein Anstieg um 172% seit Ende 2012. Mehr als drei Länder im Jahr 2013 waren Indien, die Vereinigten Staaten und Indonesien. November 2014 Opera's App Store hat begonnen, nahezu 300.000 mobile Apps und Spiele in den meisten mobilen Plattformen anzubieten. Anfang 2014 bot der mobile Speicher mehr als 200 000 Apps ein signifikantes Wachstum in der Anzahl der verfügbaren Apps an. App-Prozess Opera Mobile Store stellt keine Entwickler für die Teilnahme an dem Vertriebsprogramm und die Veröffentlichung von Apps für mobile Speicher bereit. Entwickler erhalten 70 % der Nettoeinnahmen aus dem Verkauf bezahlter Apps für Android, die im Opera Mobile Store zugelassen sind, und 50 % der Nettoeinnahmen für Java Apps. App-Genehmigungsverfahren unterliegen der Genehmigung durch das Opera Mobile Store Team für grundlegende Zuverlässigkeitsprüfungen auf allen Plattformen und Geräten, die von Entwicklern im Antragsverfahren erklärt wurden. Diese Auszeichnungen erkennen die besten Apps im Opera Mobile Store über mehrere Kategorien an, die auf der gängigen Nutzerabstimmung basieren. Partnerschaften 2013 Opera Mobile Store unterzeichnete eine länderübergreifende Vertriebspartnerschaft mit WhatsApp, der größten Suchmaschine auf dem russischen Internetmarkt und der führenden Suchmaschine in Osteuropa, basierend auf dem Publikum. Mit diesem Abkommen können Apps im Opera Mobile Store auch für das Herunterladen in der CD-App und umgekehrt zur Verfügung gestellt werden. Mit dem gleichen Jahr unterzeichneten Opera Software mehrere Partnerschaftsvereinbarungen mit Mobilfunkunternehmen in Osteuropa. Sowohl MTS Belarus als auch MTS Ukraine haben Opera co-branded App-Stores für ihre 28 Millionen gesamten Benutzerbasis eingerichtet. Operative Zeichnung Mobile Storestarting Juli 21, 2014 Opera Software hat begonnen, Dienstleistungen anzubieten, die Kunden von Mobilfunkbetreibern unbegrenzten Zugang zu Premium-Spiele und Apps bieten, die auf der Opera Mobile Store-Technologie gebaut und vom OMS-Team unterstützt werden. MTS Russland, das App Market Service zusammen mit Opera ins Leben gerufen hat, war das erste Unternehmen, das App Store-Abonnementmodell für ihre Kunden umzusetzen. Mobile Nutzer zahlen in diesem Modell eine wöchentliche Abonnementgebühr für „all-you-can-eat“-Zugang nach einer kostenlosen siebentägigen Probezeit. 2014 Opera Software startete Abonnement-Mobilgeschäfte mit MTS Russland, MTS Ukraine, MTS Belarus, TIM Brasil und XL Axiata in Indonesien.
