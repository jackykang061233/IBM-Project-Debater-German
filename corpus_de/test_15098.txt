SAP Cloud Platform ist eine von SAP SE entwickelte Plattform zur Erstellung neuer Anwendungen oder zur Erweiterung bestehender Anwendungen in einer sicheren Cloud Computing-Umgebung, die von SAP verwaltet wird. Die SAP Cloud Platform integriert Daten und Geschäftsprozesse. Übersicht SAP Cloud Platform ist eine offene Plattform-as-a-Service, die das in-memory SAP HANA Datenbank-Management-System umfasst, verbindet sich sowohl auf Räumlichkeiten als auch Cloud-basierte Systeme mit SAP oder anderen Drittanbieter-Software und setzt auf offene Standards wie Java, JavaScript, Node.js und Cloud-Gründer für Integrationsoptionen. SAP und Apple Inc. haben gemeinsam mobile Anwendungen auf iOS mit Cloud-basierten Software-Entwicklungskits (SDKs) für die SAP Cloud Platform entwickelt. SAP Cloud Platform basiert auf Open Source-Technologie, entwickelt und unterstützt in Partnerschaft mit SUSE. Das Unternehmen ist auch in Partnerschaft mit Cloud Foundry für ein Beta-Angebot von SAP Cloud Platform, das es Kunden ermöglicht, auszuprobieren und Feedback für die Funktionalitäten zu geben, die mit Cloud Foundry kommen. Die Geschichte, die ursprünglich als SAP NetWeaver Cloud zum SAP HANA Cloud-Portfolio am 16. Oktober 2012 vorgestellt wurde, wurde mit dem neuen Namen SAP HANA Cloud Platform am 13. Mai 2013 als Grundlage für SAP Cloud-Produkte, einschließlich der SAP BusinessObjects Cloud, wieder eingeführt. Die Übernahme der SAP HANA Cloud Platform ist seit dem Start der Plattform im Jahr 2012 stetig gestiegen, wobei SAP über 4000 Kunden und 500 Partner die SAP HANA Cloud Platform übernommen hat. Am 27. Februar 2017 wurde SAP HANA Cloud Platform auf dem Mobile World Congress umbenannt. Am 18. Januar 2021, SAP Cloud Platform wurde offiziell aus dem SAP-Produktportfolio entfernt, um die SAP Business Technology Platform (SAP BTP) zu unterstützen.
