Friendica (früher Friendika, ursprünglich Mistpark) ist ein kostenloses und Open-Source-Software verteiltes soziales Netzwerk. Es bildet einen Teil des Fediverse, ein vernetztes und dezentrales Netzwerk unabhängig betriebener Server. Features Friendica-Nutzer können sich mit anderen über ihren eigenen Friendica-Server verbinden, können aber auch Kontakte von anderen Plattformen wie Twitter, Diaspora, Pump.io, StatusNet und neuer ActivityPub (Mastodon, Pleroma, Pixelfed.) in ihre Newsfeed integrieren. Zusätzlich zu diesen beiden Möglichkeiten Verbindungen können Nutzer Friendica auch als Verlagsplattform nutzen, um Inhalte an WordPress, Tumblr und Libertree zu veröffentlichen. Die Buchung an Google+ wurde auch bis zum Abschalten unterstützt. Zusätzlich können E-Mail-Kontakte und RSS-Feeds integriert werden. Da Benutzer über viele Server verteilt sind, bestehen ihre Adressen aus einem Benutzernamen, dem "@" Symbol und dem Domainnamen der Friendica-Instanz in gleicher Weise E-Mail-Adressen. Die meisten Funktionen von großen Mikroblogging- und Social-Networking-Plattformen sind in Friendica verfügbar; zum Beispiel: tagging Benutzer und Gruppen über "@ Erwähnungen"; direkte Nachrichten; Hashtags; Fotoalben; Likes; Dislikes; Kommentare; und Re-shares von öffentlich sichtbaren Beiträgen. Veröffentlichte Elemente können über das Netzwerk bearbeitet und aktualisiert werden. Umfassende Einstellungen für die Privatsphäre und die öffentliche Sichtbarkeit von Beiträgen ermöglichen es Benutzern, zu regulieren, wer lesen kann, welche Beiträge oder sehen Sie bestimmte Informationen über den Benutzer. Benutzer können auch mehrere Profile erstellen, so dass verschiedene Gruppen von Personen (wie Freunde, oder Arbeitskollegen) ein anderes Profil vollständig beim Betrachten derselben Seite sehen. Benutzerkonten können heruntergeladen oder gelöscht werden und bei Bedarf zu einem anderen Friendica Server importiert werden. Öffentliche Foren können unter verschiedenen Konten erstellt werden, die zwischengeschaltet werden können, wenn die Konten mit derselben E-Mail-Adresse registriert werden. Viele Erweiterungen und Themen existieren für Friendica, einschließlich Steckverbinder, die zur Verbindung mit den oben genannten Plattformen verwendet werden können. Der Einfachheit halber werden aber auch viele neue Features als Erweiterungen geschaffen und später in den Kern von Friendica integriert. Entwicklung Es gibt keine Firma hinter Friendica. Die Entwickler arbeiten freiwillig und das Projekt wird informell betrieben, die Plattform selbst wird für die Kommunikation zwischen den Entwicklern genutzt. Es gibt verschiedene Foren in Friendica, wie "Friendica Developers" und "Friendica Support". Der Quellcode von Friendica wird auf GitHub gehostet. Installation Die Entwickler wollen die Installation der Software so einfach wie möglich für technische Laien machen. Sie argumentieren, dass die Dezentralisierung auf kleinen Servern eine zentrale Voraussetzung für die Freiheit der Nutzer und ihre Selbstbestimmung ist. Der Schwierigkeitsgrad ist ähnlich einer Installation von WordPress. Die Installation auf geteiltem Hosting ist jedoch aufgrund fehlender PHP5-Module manchmal schwierig. Einige Freiwillige betreiben auch öffentliche Server, so dass Newcomer auch die Installation ihrer eigenen Software vermeiden können. Liste der Kunden Friendica's API ist kompatibel mit der API von StatusNet/GNU social. Dadurch können die meisten GNU Social Clients für Friendica verwendet werden. Beispiele für Friendica kompatible Clients sind: Friendiqa, Fedilab, AndStatus, Tbreitee und DiCa für Android, freundlich für Sailfish OS, friclicli (CLI Client,) choqok und Friendiqa für Linux und Friendica Mobile für Windows 10. Reception Friendica wurde im Januar 2012 von Infoshop News als "Alternativ für Google+ und Facebook" zitiert, um auf der Occupy Nigeria Bewegung verwendet werden. Im Januar 2012 Der Blog der Free Software Foundation Europe zitierte Friendica als eine vernünftige Alternative zu zentralisierten und kontrollierten sozialen Netzwerken wie Facebook oder Google+.
