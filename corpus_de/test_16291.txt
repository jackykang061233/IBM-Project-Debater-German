Die Software ist auf ConsciousGrid aufgebaut, eine Technologie, die eine optimale Datenplatzierung auf Basis von Echtzeit-Metriken gewährleistet und die Zugabe oder Entfernung von Speichergeräten mit automatischer Leistungs- und Lastaufpralloptimierung ermöglicht. Für den Datenschutz OpenIO hat eine synchrone und asynchrone Replikation mit mehreren Kopien und eine Ärasure-Coding-Implementierung auf Basis von Reed-Solomon, die in einem Rechenzentrum oder geo-verteilten oder gestreckten Clustern eingesetzt werden kann. Die Software verfügt über eine GridForApps-Funktion, die alle Ereignisse erfasst, die im Cluster auftreten und im Stack oder auf OpenIO-Knoten laufenden Anwendungen übergeben kann. Dies ermöglicht ereignisgesteuertes Computing direkt in die Speicherinfrastruktur. OpenIO verfügt über native Objekt-APIs und SDKs für Python, C und Java, es integriert eine HTTP REST/API und hat eine starke Kompatibilität mit der Amazon S3 API und der OpenStack Swift API. Das Unternehmen bietet auch einen proprietären Dateisystem-Steckverbinder, um über Dateizugriffsmethoden in einem OpenIO-Objektspeicher gespeicherte Daten zuzugreifen: Es basiert auf FUSE und stellt ein POSIX File System vor, das über lokale Netzwerke über NFS, SMB und FTP geteilt werden kann. OpenIO ist kompatibel mit x86 und ARMv7/ARMv8 Servern mit Linux und hat geringe Hardwareanforderungen. Es kann auch auf Raspberry Pis und auf Speicher-Laufwerken mit Servern eingebettet installiert werden. Der Open Source Code ist auf Github verfügbar und ist unter AGPL3 für Servercode und LGPL3 für Clientcode lizenziert. Performance OpenIO behauptet, erreicht zu haben 1.372 Tbps Schreibgeschwindigkeit (171 GB/s) auf einem Cluster von 350 physikalischen Maschinen. Das Benchmark-Szenario, das unter Produktionsbedingungen mit Standard-Hardware (Compity-Server mit 7200 U/min HDDs) durchgeführt wurde, bestand darin, einen 38 PB Hadoop Datalake über den DistCp-Befehl zu sichern. Diese Leistungsstufe markierte laut Analysten die Ankunft einer neuen Generation von Objektspeichertechnologien, die auf eine hohe Leistung und Hyperskalierbarkeit ausgerichtet ist.
