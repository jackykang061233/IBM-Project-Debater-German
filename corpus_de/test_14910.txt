Opera ist ein multi-platformer Web-Browser entwickelt von Opera Software. Opera ist ein Chromium-basierter Browser. Es unterscheidet sich von anderen Browsern durch seine Benutzeroberfläche und andere Funktionen. Opera wurde ursprünglich im April 1995 veröffentlicht, so dass es einer der ältesten Desktop-Webbrowser noch aktiv entwickelt heute. Es war eine kommerzielle Software für die ersten zehn Jahre und hatte eine eigene proprietäre Presto Layout-Engine. Im Jahr 2013 wechselte Opera von der Presto-Engine auf Chrom. Der Webbrowser kann unter Microsoft Windows, Android, iOS, macOS und Linux Betriebssystemen verwendet werden. Es gibt auch mobile Versionen namens Opera Mobile und Opera Mini. Darüber hinaus haben Opera-Nutzer Zugang zu einer Nachrichten-App basierend auf einer AI-Plattform, Opera News. Das Unternehmen veröffentlichte einen Gaming-Browser namens Opera GX in 2019. Entwicklungsstadien Opera Software verwendet einen Release-Zyklus bestehend aus drei Streams, die den Phasen der Entwicklung entsprechen, die unabhängig voneinander heruntergeladen und installiert werden können: Entwickler, Beta und stabil". Neue Funktionen werden zunächst in der Entwickler-Bauweise vorgestellt, dann, je nach Benutzer-Feedback, kann Fortschritte auf die Beta-Version und schließlich veröffentlicht werden. Der Entwickler-Stream ermöglicht eine frühe Prüfung neuer Funktionen, vor allem auf Entwickler, Erweiterungs-Ersteller und frühen Adopter. Opera-Entwickler ist nicht für den täglichen Surfen bestimmt, da es instabil ist und ist anfällig für Fehler oder Absturz, aber es ermöglicht fortgeschrittenen Benutzern, neue Funktionen auszuprobieren, die noch in der Entwicklung sind, ohne ihre normale Installation des Browsers zu beeinflussen. Neue Versionen des Browsers werden häufig veröffentlicht, in der Regel ein paar Mal pro Woche. Der Beta-Strom, früher als "Opera Next" bekannt, ist ein komplettes Paket, das Stabilität und Qualität vor der endgültigen Freigabe reifen lässt. Eine neue Version wird alle paar Wochen veröffentlicht. Beide Streams können neben dem offiziellen Release ohne Störung installiert werden.
