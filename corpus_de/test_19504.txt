In Ökonometrie bezieht sich die Endogenität weitestgehend auf Situationen, in denen eine erläuternde Größe mit dem Fehlerterm korreliert. Die Unterscheidung zwischen endogenen und exogenen Variablen entstand in simultanen Gleichungsmodellen, wobei man Variablen trennt, deren Werte vom Modell aus vorgegebenen Variablen bestimmt werden; die ignorierende Simultanität in der Schätzung führt zu voreingenommenen Schätzungen, da sie die Exogenitätsannahme des Gauss-Markov-Theorems verletzt. Das Problem der Endogenität wird oft von Forschern, die nicht-experimentelle Forschung betreiben, ignoriert und damit politische Empfehlungen ausschließen. Instrumentalvariable Techniken werden häufig verwendet, um dieses Problem zu lösen. Neben der Simultanität kann eine Korrelation zwischen erläuternden Variablen und dem Fehlerterm entstehen, wenn eine unbeobachtete oder weggelassene Variable sowohl unabhängige als auch abhängige Variablen enthält oder wenn unabhängige Variablen mit Fehler gemessen werden. Exogenese gegen Endogenität In einem stochastischen Modell kann der Begriff der üblichen Exogenität, sequentielle Exogenität, starke/strikte Exogenität definiert werden. Die Exogenität ist derart artikuliert, dass für den Parameter α {\displaystyle \alpha } eine Variable oder Variable exogen ist. Auch wenn eine Variable exogen für den Parameter α {\displaystyle \alpha } ist, kann es für den Parameter β {\displaystyle \beta } endogen sein.Wenn die erläuternden Variablen nicht stochastisch sind, dann sind sie stark exogen für alle Parameter. Wird die unabhängige Variable mit dem Fehlerterm in einem Regressionsmodell korreliert, so wird die Schätzung des Regressionskoeffizienten in einer gewöhnlichen Mindestquadrate (OLS)-Regression vorgespannt, wenn die Korrelation jedoch nicht kontemporan ist, so kann die Koeffizientenschätzung noch konsistent sein. Es gibt viele Methoden zur Korrektur der Vorspannung, einschließlich instrumental variable Regression und Heckman Auswahlkorrektur. Dynamische Modelle Das Endogenitätsproblem ist insbesondere bei der Zeitreihenanalyse von Kausalprozessen relevant. Es ist üblich, dass einige Faktoren innerhalb eines Kausalsystems für ihren Wert im Zeitraum t von den Werten anderer Faktoren im Kausalsystem im Zeitraum t -1 abhängig sind. Angenommen, die Höhe des Schädlingsbefalls ist unabhängig von allen anderen Faktoren innerhalb eines bestimmten Zeitraums, wird aber durch die Höhe von Niederschlägen und Düngemitteln in der vorangegangenen Periode beeinflusst. In diesem Fall wäre es richtig zu sagen, dass der Befall innerhalb der Zeit exogen, aber im Laufe der Zeit endogen ist. Lassen Sie das Modell y = f(x, z) + u sein. Ist die Variable x sequentielle exogen für den Parameter α {\displaystyle \alpha } und y verursacht im Granger-Sense nicht x, so ist die Variable x für den Parameter α {\displaystyle \alpha } stark/einschnürlich exogen.
