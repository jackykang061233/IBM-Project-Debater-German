In der Mathematik ist eine kombinatorische Explosion das schnelle Wachstum der Komplexität eines Problems, da die Combinatorik des Problems durch die Eingabe, Zwänge und Grenzen des Problems beeinflusst wird. Die kombinierte Explosion wird manchmal verwendet, um die Anlenkbarkeit bestimmter Probleme zu rechtfertigen. Beispiele für solche Probleme sind bestimmte mathematische Funktionen, die Analyse einiger Rätsel und Spiele und einige pathologische Beispiele, die als Ackermann-Funktion modelliert werden können. Stellen Sie sich ein einfaches System mit nur einer Variable vor, ein Boolean namens A.Das System hat zwei mögliche Zustände, A = true oder A = false. Hinzufügen einer anderen booleanischen Variable B wird dem System vier mögliche Zustände geben, A = true und B = true, A = true und B = false, A = false und B = true, A = false und B = false. Ein System mit n booleans hat 2n mögliche Zustände, während ein System von n Variablen mit je Z erlaubten Werten (statt nur die 2 (wahr und falsch) von booleans) Zn mögliche Zustände haben wird. Die möglichen Zustände können als Blattknoten eines Baumes der Höhe n betrachtet werden, wo jeder Knoten Z Kinder hat. Diese rasche Zunahme der Blattknoten kann in Bereichen wie der Suche nützlich sein, da viele Ergebnisse zugegriffen werden können, ohne sehr weit absteigen zu müssen. Es kann auch ein Hindernis sein, wenn solche Strukturen manipuliert werden. Eine Klassenhierarchie in einer objektorientierten Sprache kann als Baum gedacht werden, wobei verschiedene Arten von Objekten von ihren Eltern erben. Sollen verschiedene Klassen kombiniert werden, wie z.B. im Vergleich (wie A < B), dann explodiert die Anzahl der möglichen Kombinationen, die auftreten können. (Dieses Wachstum wird oft lässig als exponentiell beschrieben, ist aber eigentlich polynomial.) Wenn zwei Organisationen über ein bestimmtes Thema kommunizieren müssen, kann es am einfachsten sein, direkt ad-hoc zu kommunizieren - es ist nur ein Kommunikationskanal erforderlich. Wenn jedoch eine dritte Organisation hinzugefügt wird, sind drei separate Kanäle erforderlich. Die Aufnahme einer vierten Organisation erfordert sechs Kanäle; fünf, zehn; sechs, fünfzehn; usw.
