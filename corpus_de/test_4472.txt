In Computing ist ein Server ein Stück Computer-Hardware oder -Software (Computerprogramm), das Funktionalitäten für andere Programme oder Geräte, sogenannte Kunden, bietet. Diese Architektur wird als Kunden-Server-Modell bezeichnet. Server können verschiedene Funktionalitäten anbieten, häufig als Dienste, wie die Weitergabe von Daten oder Ressourcen bei mehreren Kunden oder die Ausführung einer Berechnung für einen Kunden. Ein einzelner Server kann mehrere Kunden bedienen, und ein einzelner Kunde kann mehrere Server verwenden. Ein Kunde-Prozess kann auf dem gleichen Gerät laufen oder über ein Netz an einen Server auf einem anderen Gerät verbinden. Typische Server sind Datenbankenserver, Dateiserver, Mailserver, Druckserver, Webserver, Spielserver und Anwendungsserver. Kunden-Server-Systeme werden heute am häufigsten von (und oft mit) dem Anforderungs- und Antwortmodell umgesetzt: ein Kunde sendet einen Antrag an den Server, der eine Maßnahme durchführt und dem Kunden eine Antwort übermittelt, in der Regel mit einem Ergebnis oder einer Anerkennung.
