Es wird verwendet, um das Konzept der steuerbaren Einkommenlastizität zu illustrieren (d. h. das steuerpflichtige Einkommen wird sich entsprechend der Steuervergünstigung ändern). Die Kurve wird durch Denkversuch gebaut. Erstens wird die Höhe der Steuereinnahmen, die auf den extremen Steuersätzen von 0% und 100% erhoben werden, berücksichtigt. Klar ist, dass ein Steuersatz von 0 % keine Einnahmen aufwirft, aber die Laffer-Kurvenhypothesis ist, dass eine 100%ige Steuerquote auch keine Einnahmen generieren wird, da es für einen rationalen Steuerzahler keinen Anreiz mehr gibt, ein Einkommen zu verdienen, so dass die aufgebrachten Einnahmen 100 % des Nichts ausmachen. Liegt sowohl ein Steuersatz von 0 % als auch eine 100%-ige Steuerquote keine Einnahmen, so ergibt sich daraus der extreme Wert, den der Begünstigte zumindest einen Satz zwischen dem, in dem die Steuereinnahmen einen Höchstsatz darstellen. Die Laffer-Kurve ist in der Regel als Graphik vertreten, die bei 0 % Steuern, Nulleinnahmen beginnt, zu einem maximalen Steuersatz führt und dann wieder zu Nulleinnahmen in einer 100 %igen Steuersatz.
