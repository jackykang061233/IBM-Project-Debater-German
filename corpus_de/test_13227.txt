Microsoft Edge ist ein plattformübergreifender Webbrowser, der von Microsoft erstellt und entwickelt wurde. Es wurde erstmals veröffentlicht für Windows 10 und Xbox One in 2015, für Android und iOS in 2017, für macOS in 2019, und als Vorschau für Linux im Oktober 2020, und kann Internet Explorer unter Windows 7, Windows 8, Windows 8.1, Windows Server 2008 R2, Windows Server 2012, Windows Server 2012 R2, Windows Server 2016 und Windows Server 2019 ersetzen, aber im Gegensatz zu IE, dieser Browser unterstützt Windows Vista nicht oder eine frühere Version. Edge wurde zunächst mit dem eigenen proprietären Browser-Engine von Microsoft EdgeHTML und der Chakra JavaScript-Engine, einer Version, die jetzt als Microsoft Edge Legacy bezeichnet wird, gebaut. Im Jahr 2019 kündigte Microsoft an, den Browser als Chromium-basierte mit Blink- und V8-Motoren wieder aufzubauen. Während der Entwicklung (codiert Anaheim,) Microsoft machte Vorschau baute von Edge auf Windows 7, 8/8.1, 10 und macOS zur Verfügung. Microsoft hat am 15. Januar 2020 die Veröffentlichung des neuen Edge angekündigt. Im Juni 2020 startete Microsoft das automatische Rollout der neuen Version über Windows Update für Windows 7, 8.1, und Windows 10 Versionen von 2003 bis 2004. Microsoft stoppte die Freigabe von Sicherheits-Patches für Edge Legacy vom 9. März 2021 und veröffentlichte ein Sicherheits-Update am 13. April 2021, die Edge Legacy ersetzt durch Chrom-basierte Edge. Microsoft veröffentlichte die Chromium-basierte Edge auf die Xbox Insider Alpha Skip Ahead Gruppe am 6. März 2021. Microsoft Edge ist der Standard-Webbrowser unter Windows 10, Windows 10 Mobile, Windows 11, Xbox One und Xbox Series X und Series S Konsolen, ersetzt Internet Explorer 11 und Internet Explorer Mobile. Da seine Entwicklung und Veröffentlichung von dem Modell von Windows als Dienst abhängig ist, ist es nicht in Windows 10 Enterprise Long-Term Servicing Channel (LTSC) erstellt enthalten. Microsoft kündigte zunächst an, dass Edge den legacy Trident (MSHTML)-Layout-Engine für Rückwärtskompatibilität unterstützen würde, aber später sagte, dass Edge aufgrund "starkes Feedback" einen neuen Motor verwenden würde, während Internet Explorer den legacy-Engine weiter bereitstellen würde. Favoriten, Leseliste, Browser-Geschichte und Downloads werden am Hub betrachtet, eine Sidebar, die Funktionen wie Internet Explorer Downloads Manager und Favorites Center bietet. Edge verfügt über einen PDF-Reader und unterstützt asm.js. Bis Januar 2021 verfügte Edge auch über einen integrierten Adobe Flash Player (mit einer internen Whitelist, die Flash-Applets auf Facebook-Websites automatisch laden lässt, um alle anderen Sicherheitskontrollen, die eine Benutzeraktivierung erfordern, zu umgehen). Edge unterstützt nicht ältere Technologien wie ActiveX und Browser Helper Objects, sondern verwendet ein Erweiterungssystem. Internet Explorer 11 bleibt neben Edge unter Windows 10 für Kompatibilität zur Verfügung; es bleibt identisch mit der Windows 8.1 Version und verwendet nicht die Edge-Engine, wie zuvor angekündigt. Edge integriert sich mit Microsofts Online-Plattformen, um Sprachsteuerung, Suchfunktionalität und dynamische Informationen zu suchen innerhalb der Adressleiste. Benutzer können Annotationen auf Webseiten machen, die auf OneDrive gespeichert und freigegeben werden können und HTML- und MHTML-Seiten auf ihren Computern speichern können. Es integriert sich auch mit der Funktion "Reading List" und bietet einen "Reading Mode", der unnötige Formatierung von Seiten zur Verbesserung ihrer Lesbarkeit abstreift. Vorläufige Unterstützung für Browser-Erweiterungen wurde im März 2016 hinzugefügt, mit Build 14291, drei Erweiterungen wurden ursprünglich unterstützt. Microsoft gab an, dass die Verzögerung bei der Genehmigung von Erweiterungen und die kleine Zahl auf Sicherheitsbedenken zurückzuführen war. ENTWICKLUNG EdgeHTML wurde ursprünglich für Edge entwickelt. Es war eine Gabel von Trident, die alle Vermächtniscode von älteren Versionen von Internet Explorer entfernt, mit der Mehrheit seiner Quellcode neu geschrieben, um Web-Standards und Interoperabilität mit anderen modernen Browsern zu unterstützen. EdgeHTML wurde in C++ geschrieben. Die Rendering-Engine wurde erstmals als experimentelle Option im Internet Explorer 11 im Rahmen der Windows 10 Preview 9926 Build veröffentlicht. EdgeHTML sollte vollständig kompatibel mit der WebKit Layout-Engine von Safari, Chrome und anderen Browsern verwendet werden. Microsoft erklärte ihre ursprünglichen Akzeptanzkriterien: "Any Edge–WebKit Unterschiede sind Bugs, die wir an der Fixierung interessiert sind. " Eine Überprüfung des Motors in der Beta Windows 10 von AnandTech gefunden erhebliche Benchmark Verbesserungen über Trident, vor allem JavaScript-Engine-Leistung, die bis zu par mit der von Google Chrome. Andere Benchmarks, die sich auf die Leistung der WebGL API konzentrierten, fanden EdgeHTML, um viel besser als Google Chrome und Mozilla Firefox durchzuführen. HTML5 Standards Edge hatte ursprünglich keine Unterstützung für offene Medienstandards wie WebM und Opus, aber diese wurden später in Edge 14.14291 hinzugefügt. Die EdgeHTML-Version von Microsoft Edge bleibt aus Kompatibilitätsgründen installiert, aber Windows wird es verstecken (Version 44.19041.1.0). Ab August 2020 hatte Edge 84 496/555 auf HTML5test erzielt. Anfang Februar 2016 wurde der Bug mit einem kumulativen Update am 9. Februar 2016 behoben. Microsofts Wechsel zu Blink, da Edges Motor gemischten Empfang konfrontiert. Die Bewegung erhöht die Konsistenz der Web-Plattform-Kompatibilität zwischen großen Browsern. Aus diesem Grund hat die Bewegung Kritik angezogen, da es die Vielfalt im gesamten Web-Browser-Markt reduziert und den Einfluss von Google (Entwickler der Blink-Layout-Engine) auf den gesamten Browser-Markt von Microsoft ceding seine unabhängig entwickelte Browser-Engine erhöht. Laut Douglas J Leith, einem Informatik-Professor von Trinity College, Dublin, gehört Microsoft Edge zu den am wenigsten privaten Browsern. Als Antwort erklärte ein Sprecher von Microsoft Edge, dass es Benutzerdiagnosedaten verwendet, um das Produkt zu verbessern. Im Juni 2020 kritisierten Benutzer neu veröffentlichte Windows 7, Windows 8.1 und Windows 10 Updates, die Edge installiert und importierte einige Benutzerdaten von Chrome und Firefox, bevor Sie Benutzerberechtigung. Microsoft reagierte darauf, dass, wenn ein Benutzer das Geben von Edge-Datenimport-Berechtigung zurückweist, Edge die importierten Daten löschen wird. Wenn jedoch der Browser abstürzt, bevor der Nutzer die Möglichkeit hat, den Import abzulehnen, werden die bereits importierten Daten nicht gelöscht. Die Verge nannte diese "Spyware-Taktik" und nannte Edges "erste Lauferfahrung" ein "dunkles Muster". Marktanteil Nach StatCounter, im August 2019 Edge übernahm den Marktanteil von Internet Explorer (IE) auf PC, Edge auf Platz vier und IE in fünf. Während IE-Aktien fallen, ist keine einzige Version von Edge beliebter als Internet Explorer 11. Der Marktanteil für Edge bleibt niedrig, wobei IE in diesem Trend folgt. Mobile Versionen von Edge existieren für Android und iOS, aber sie haben wenig bis keinen Marktanteil. Auf Microsoft-Konsolen ersetzte Edge IE als dominanter Browser einige Monate nach seiner Veröffentlichung im Jahr 2015. Der Marktanteil variiert je nach Region. An einigen Tagen der Woche, Edge findet zweiten Platz mit einem 10.02% Anteil in den USA auf dem PC, und Firefox und Edge haben sehr ähnliche Anteil weltweit, Schalter Plätze für zweiten und dritten Rang je nach Tag. Zum Beispiel, im März 2020, Edge rangierte die zweite mit Marktanteil von 7.59,% überholen Firefox, die hatte 7.19% des Marktanteils.
