Windows Phone 7 ist die erste Veröffentlichung des Windows Phone mobilen Client-Betriebssystems, weltweit am 21. Oktober 2010 veröffentlicht, und in den Vereinigten Staaten am 8. November 2010. Es läuft auf dem Windows CE 6.0 Kernel. Es erhielt mehrere große Updates, das letzte ist Windows Phone 7.8, die im Januar 2013 veröffentlicht wurde und fügte ein paar Funktionen zurückportiert von Windows Phone 8, wie ein anpassbarer Startbildschirm. Microsoft beendete die Unterstützung für Windows Phone 7 am 14. Oktober 2014. Es wurde von Windows Phone 8 gefolgt, die am 29. Oktober 2012 veröffentlicht wurde. Historie Microsoft hat am 15. Februar 2010 das neue Betriebssystem Windows Phone 7 Series auf dem Mobile World Congress in Barcelona offiziell vorgestellt und am 15. März 2010 weitere Details auf der MIX 2010 vorgestellt. Das letzte SDK wurde am 16. September 2010 zur Verfügung gestellt. Funktionen Core Windows Phone 7 ist die einzige Version von Windows Phone, die einen Kernel basierend auf der Windows Embedded Compact 7 Version von Windows Embedded CE, die auch in Windows Mobile und Pocket PC Systemen verwendet wurde. Die Benutzeroberfläche Windows Phone 7 verfügt über eine Benutzeroberfläche, die auf einem Designsystem basiert, das codiert und allgemein als Metro bezeichnet wird. Der Home-Bildschirm, genannt "Start-Bildschirm", besteht aus "Live Tiles". Fliesen sind Links zu Anwendungen, Features, Funktionen und einzelnen Elementen (wie Kontakte, Webseiten, Anwendungen oder Medienartikel). Benutzer können Fliesen hinzufügen, umordnen oder entfernen. Fliesen sind dynamisch und aktualisieren in Echtzeit – zum Beispiel würde die Kachel für ein E-Mail-Konto die Anzahl der ungelesenen Nachrichten oder eine Kachel ein Live-Update des Wetters anzeigen. Mehrere Schlüsselfunktionen von Windows Phone 7 werden in Hubs organisiert, die lokale und Online-Inhalte über die Integration von Windows Phone mit beliebten sozialen Netzwerken wie Facebook, Windows Live und Twitter kombinieren. Zum Beispiel zeigt der Pictures-Hub mit der Kamera des Geräts und den Facebook-Fotoalben des Benutzers aufgenommene Fotos, und der People-Hub zeigt Kontakte aus mehreren Quellen, einschließlich Windows Live, Facebook und Gmail. Aus dem Hub können Benutzer direkt kommentieren und wie auf Social Network-Updates. Die anderen integrierten Hubs sind Xbox Music und Video, Xbox Live Games, Windows Phone Store und Microsoft Office. Die standardmäßige Windows Phone Benutzeroberfläche hat ein dunkles Thema, das die Batterielebensdauer auf OLED-Bildschirmen verlängert, da voll schwarze Pixel kein Licht emittieren. Der Benutzer kann stattdessen ein Lichtthema wählen und kann auch aus mehreren Akzentfarben wählen. App-Entwickler (beide Inhouse und ISV) können verschiedene Versionen der virtuellen Tastatur angeben, um Benutzer auf bestimmte Zeichensätze zu beschränken, wie numerische Zeichen allein. Die Hardwareanforderungen von Microsoft stellen fest, dass jedes Gerät, das Windows Phone 7 betreibt, eine dedizierte Suchtaste auf der Vorderseite des Geräts haben muss, das verschiedene Aktionen durchführt. Durch Drücken der Suchtaste, während eine Anwendung geöffnet ist, können Benutzer innerhalb von Anwendungen suchen, die diese Funktion nutzen; zum Beispiel durch Drücken von Search in the People hub können Benutzer ihre Kontaktliste für bestimmte Personen suchen.
