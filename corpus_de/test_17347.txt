Jelastic ist ein Cloud-Plattform-Software-Anbieter, der Multi-Cloud PaaS (Plattform als Service) bietet, basierend auf der Containertechnik für Anbieter von Dienstleistungen, ISVs, Telekommunikationsunternehmen, Unternehmen und Entwickler. Die Plattform ist als öffentliche Cloud in mehr als 70 Datenzentren weltweit, privater Cloud (virtuelles und vor Ort), Hybrid- und Multi-Cloud verfügbar. Jelastic unterstützt Java, PHP, Ruby, Node.js, Python, Goumgebungen, Zollcontainer und Kubernetes Cluster. Jelastics Mitgründer und derzeitiger CEO ist Ruslan Synytskyy. Jelastic wurde 2011 als öffentliche Cloud gegründet. Anfangs war ein PaaS speziell auf Java ausgerichtet. 2012 erhielt das Unternehmen Java Herzog Choice Award. 2013-2015 ergänzte die Plattform die Unterstützung von PHP, Ruby, Python, Node.js, .NET und Unterstützung von Docker Containern. Im Jahr 2017 war die letzte Laufzeit durch die Integration von GoLang. 2014 wurde eine private Cloud-Lösung eingeführt, und 2015 könnte die Plattform bereits als Hybrid- und Multi-Cloud genutzt werden. Jelastic schloss eine Reihe von Finanzierungsrunden von Runa Capital, Almaz Capital Partners und Foresight Ventures und Maxfield Capital. Jelastic kündigte im Juni 2013 die Einstellung von Mark Zbikowski, einem ehemaligen Microsoft-Ingenieur und einem Beitrag zu MS-DOS, als Seniorberater an. Im Juli 2013 trat Rasmus Lerdorf, Autor von PHP, als Seniorberater bei. Im November 2014 traten die Autoren der Java-Programmsprache James Gosling und der brasilianischen Java Man Bruno Souza dem Jelastic Team bei. Jelastic hält an der Verbesserung der Java-Technologie und der Nutzererfahrung fest, so dass die Plattform 2018 ihre führende Position in dieser Richtung bewiesen hat, die den Wahlpreis von Herzog zum zweiten Mal erhielt. Jelastic fügte im März 2019 die Unterstützung von Kubernetes-Diensten hinzu, die vorverpackte Cluster anbieten, die automatisch installiert und kaliert werden können. Dienstleistungen unterstützende Technologien Sprachen:Java, PHP, Ruby, Node.js, Python, .NET, Go Virtualization: Docker (Software), Virtuozzo Load-Balancers: NGINX, Apache, HAPoxy, Varnish, LiteSpeed-Anwendungen: Tomcat, LiteSpeed, GlassFish, Jetty, Payara, NGINX PHP, NIPSX Semiconductors, SmartFoxServer, Railo, Frühling, Tom, WildFly, JBoss-Datenbanken: Marian, MySQL, Universal, Mongo4 DB VPS: CentOS, Ubuntu, Windows VPS Zusatz: Maven, Memcaned, Storage Integrated IDEs: FauJ IDEA, Skype, NetBeans Installation Die Plattform kann auf reinem Metall oder beliebigen IaaS als öffentliche Cloud, VPC, Privat- und Hybrid- oder Multi-Cloud installiert werden. Jelastic PaaS kann von Jelastic Ops Team installiert und voll ausgestaltet werden, oder dies kann automatisch zusätzlich zu Google Cloud, MS Azure und Digital Ocean sowie zusätzlich zu reinem Metall mit einem ISO-Bild durchgeführt werden. Die Plattformkonfiguration umfasst die Integration mit Abrechnungssystemen wie WHMCS, Odin System Automation, Cleverbridge oder kundenspezifische Abrechnung, falls erforderlich. Preismodell Jelastic bietet Pay-as-you-use Preismodell in der öffentlichen Cloud. Ressourcen werden auf stündlicher Basis berechnet: RAM und CPU (Cloudlet Resources Unit), Diskraum, Außenverkehr, Public IP und SSL. Der Preis für jede Ressourceneinheit hängt von dem gewählten Hosting-Anbieter ab und kann auf einer monatlichen, täglichen oder stündlichen Basis innerhalb des Dev-Panels geschätzt werden. Cloud-Hosting-Ausgaben können mit eingebauten Abrechnungsangaben für das gesamte Konto oder eine spezifische Anwendung verfolgt und optimiert werden. Private Cloud-Installation wird monatlich auf einer Lizenzbasis pro physischen Server berechnet. Es gibt zwei Arten von Lizenzen - für die Installation mit weiterer Selbstverwaltung und für die Installation mit verwalteten Diensten. Partnerschaft mit Diensteanbietern wird auf dem Franchise-Modell mit Einnahmenteilung aufgebaut. Partner Jel Partnersastic wird von einer Vielzahl von Anbietern und Telcos installiert, darunter DataCenter Finnland, Telecom Italia, eApps, Infomaniak, Premierminister Tel, Schichtwechsel, Elastx, Innofield, Aruba Cloud, Hidora, CloudJiffy und andere. Kunden Jelastic wird von Systemintegratoren und Unternehmen aus verschiedenen Bereichen (Finanzierung, Software, Sport, Spiel usw.) wie FA Solutions, International Football Association Board, GMV, G5 Spiele, Miele, Philips Lighting und andere genutzt.
