Eine elektronische Mail-Liste oder E-Mail-Liste ist eine besondere Nutzung von E-Mail, die vielen Internetnutzern eine breite Verbreitung von Informationen ermöglicht. Es ist ähnlich wie eine traditionelle Mailliste – eine Liste von Namen und Adressen – wie sie von einer Organisation für die Übermittlung von Veröffentlichungen an ihre Mitglieder oder Kunden gehalten werden könnten, in der Regel aber auf vier Dinge: eine Liste von E-Mail-Adressen, die Personen (Teilnehmer) die Post an diesen Adressen erhalten, so dass eine Gemeinschaft rund um ein Thema von Interesse definiert wird. die Veröffentlichungen (E-Mails), die an diese Adressen gesendet werden, und ein Reflektor, eine einzige E-Mail-Adresse, die, wenn sie als Empfänger einer Nachricht benannt werden, eine Kopie dieser Botschaft an alle Abonnenten senden wird. System Electronic Mail Listen sind in der Regel vollständig oder teilweise automatisiert durch die Verwendung von speziellen Mail-Liste-Software und einer auf einem Server eingerichteten Reflektionsadresse. Künftige Nachrichten, die an die Kontoor-Adresse gesendet werden, werden von der Software verarbeitet und werden je nach Inhalt intern gehandelt (im Falle von Nachrichten, die die Software selbst enthalten) oder an alle E-Mail-Adressen verteilt, die auf die Mail-Liste abgestimmt sind. Diskussionsliste Eine weitere Art der elektronischen Mailliste ist eine Diskussionsliste, in der jeder Teilnehmer einen Post-Auftrag abstellen kann. In einer Diskussionsliste verwendet ein Teilnehmer die Mailliste, um Botschaften an alle anderen Abonnenten zu senden, die in ähnlicher Weise beantworten können. So können echte Diskussionen und Informationsaustausch stattfinden. Mailing Listen dieser Art sind in der Regel thematisch ausgerichtet (z.B. Politik, wissenschaftliche Diskussion, Gesundheitsprobleme, Hörspielwettbewerbe), und das Thema kann von äußerster Nähe bis "Was Sie für uns interessieren könnten" reichen.
