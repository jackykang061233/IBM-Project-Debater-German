Windows Phone 8.1 ist die dritte Generation von Microsofts Windows Phone mobile Betriebssystem, gefolgt von Windows Phone 8. Auf der Microsoft's Build Conference in San Francisco, Kalifornien, am 2. April 2014, wurde es am 14. April 2014 in letzter Form veröffentlicht und erreichte am 4. August 2014 die allgemeine Verfügbarkeit. Alle Windows Phones mit Windows Phone 8 können auf Windows Phone 8.1 aktualisiert werden, mit Veröffentlichung abhängig von den Daten des Trägers Rollout. Windows Phone 8.1 ist auch die letzte Version, die den Windows Phone-Markennamen verwendet, wie es von Windows 10 Mobile gelang. Einige Windows Telefon 8.1 Geräte können auf Windows 10 Mobile aktualisiert werden. Microsoft verzögerte das Upgrade und reduzierte die unterstützte Geräteliste von ihrem ersten Versprechen. Der Support wurde am 11. Juli 2017 für Windows Phone 8.1 beendet. Geschichte Windows Phone 8.1 wurde zuerst gerübt, um Windows Phone Blue zu sein, eine Reihe von Updates zu Microsoft Handy-Betriebssystem, die mit der Veröffentlichung von Windows 8.1. Obwohl Microsoft ursprünglich geplant hatte, WP8.1 Ende 2013 zu veröffentlichen, kurz nach der Freigabe seines PC-Gegenstands, wurde die allgemeine Verteilung des neuen Betriebssystems bis Anfang 2014 zurückgeschoben. Anstatt über ein Jahr warten, um neue Funktionen zu Windows Phone 8 hinzuzufügen, entschied sich Microsoft, drei inkrementelle Updates zu seinem bestehenden mobilen Betriebssystem zu veröffentlichen. Diese Updates werden mit entsprechenden Firmware-Updates für die jeweiligen Geräte geliefert. Die Updates enthalten DDR2 (Lumia Amber), die Funktionen wie "Data Sense", und DDR3 (Lumia Black), die Unterstützung für Quad-Core-Prozessoren, 1080p High-Definition-Bildschirme von bis zu sechs Zoll, die Hinzufügung eines "Driving Mode", und zusätzliche Reihen von Live-Fliesen für größere Phablet-Geräte. Der letzte Name des aktualisierten Betriebssystems wurde an die Öffentlichkeit weitergegeben, als Microsoft die Windows Phone 8.1 SDK an Entwickler am 10. Februar 2014 veröffentlichte, aber es war nicht bis Microsofts Build Konferenz Keynote am 2. April 2014, als Windows Phone 8.1 offiziell bekannt gegeben wurde, neben dem Windows 8.1 Update. Der endgültige Versandcode wurde an registrierte Benutzer des Programms "Vorschau für Entwickler" am 14. April 2014 und an die breite Öffentlichkeit in den folgenden Monaten veröffentlicht, wobei das tatsächliche Veröffentlichungsdatum von den kabellosen Trägern der Geräte und begleitet von Firmware-Updates, einschließlich Lumia Cyan. Vorschau für Entwickler Das Programm "Vorschau für Entwickler" wurde im Oktober 2013 mit der Veröffentlichung von Windows Phone 8 Update 3 initiiert. Das Programm war für Entwickler und Enthusiasten gedacht, um sofortigen Zugriff auf OS-Updates zu gewinnen, da sie von Microsoft zur Verfügung stehen, um drahtlose Träger und OEMs, die Änderungen testen, bevor einschließlich gerätespezifische Firmware-Updates. Benutzer des Programms "Vorschau für Entwickler" nicht ihre Garantie in den meisten Fällen und kann jede zukünftige Firmware installieren, die mit dem offiziellen Rollout des Windows Phone 8.1. Die Windows Telefon-Software-Updates, die über "Vorschau für Entwickler" geliefert werden, sind komplett, fertige Versionen des Betriebssystems, im Gegensatz zu den Windows 10 Mobile Builds im Windows Insider-Programm, die Vorschau-Versionen der Software sind, die für Benutzer gedacht sind, um neue Funktionen vor der endgültigen Version auszuprobieren und kann Bugs enthalten. Features Windows Phone 8.1 stellt eine Vielzahl von bemerkenswerten neuen Funktionen vor, von denen die meisten in einer Vorschau veröffentlicht, um Entwickler am 10. Februar 2014. Cortana Cortana ist ein persönlicher virtueller Assistent, der in Windows Phone 8.1 hinzugefügt wurde und ähnlich wie Google Now und Apples Siri ist. Der Cortana-Name leitet sich von der Halo-Videospiel-Serie ab, die ein Microsoft-Franchise exklusiv für Xbox und Windows ist. Cortanas Funktionen umfassen die Möglichkeit, Erinnerungen einzustellen, natürliche Stimme zu erkennen, ohne dass der Benutzer eine vordefinierte Reihe von Befehlen eingeben muss und Fragen mit Informationen von Bing (wie aktuelle Wetter- und Verkehrsbedingungen, Sportpunkte und Biographien) beantworten muss. Cortana verwendet auch ein spezielles Feature namens ein Notebook, wo es automatisch Informationen über und Interessen des Benutzers basierend auf der Nutzung sammeln und dem Benutzer ermöglichen, zusätzliche persönliche Informationen einzugeben, wie ruhige Stunden und enge Freunde, die während dieser ruhigen Stunden durch den Benutzer gelangen können. Benutzer können auch Informationen aus dem Notebook löschen, wenn sie es für Cortana zu wissen unerwünscht halten. Windows 8.1 universell Bing SmartSearch-Funktionen werden in Cortana integriert, die die vorherige Bing Search-App ersetzt, die aktiviert wird, wenn ein Benutzer die Such-Taste auf seinem Gerät drückt.Diese Funktion, die derzeit in Beta ist, wurde in den Vereinigten Staaten im ersten Halbjahr 2014 und in China, dem Vereinigten Königreich, Indien, Kanada und Australien im August 2014 veröffentlicht. Microsoft hat sich verpflichtet, Cortana zweimal im Monat zu aktualisieren und Funktionen hinzuzufügen. Die neuen Funktionen können mehr "Ost-Ei" Antworten, Verbesserungen in UI und bessere Sprachmodulationen enthalten. Windows Telefon 8.1 verwendet eine mobile Version des Internet Explorer 11 als Standard-Webbrowser. IE11 übernimmt viele der Verbesserungen des Desktop-Gegenstands, darunter Unterstützung für WebGL, normales Mapping, InPrivate-Modus, Lesemodus und die Fähigkeit, links oder rechts zu schweben, um auf eine vorherige Webseite und zurück zu navigieren. Der aktualisierte Browser enthält auch einen neuen HTML5-Video-Web-Player mit Unterstützung für Inline-Wiedergabe und geschlossene Untertitel, Windows 8-Stil Website Live-Fliesen und die Möglichkeit, Passwörter zu speichern. Darüber hinaus können Benutzer jetzt eine unbegrenzte Anzahl von Tabs öffnen, anstatt das vorherige Maximum von 6. Wenn ein Benutzer mit seinem Microsoft-Account sowohl auf ihrem Windows 8.1 Gerät als auch auf Windows Phone eingeloggt ist, werden seine Tabs auf IE11 nun automatisch synchronisiert. Apps und Windows Phone Store App Framework Apps für Windows Phone 8.1 können jetzt mit dem gleichen Anwendungsmodell wie Windows Store Apps für Windows 8.1, basierend auf der Windows Runtime erstellt werden, und die Dateierweiterung für WP Apps ist jetzt .appx (die für Windows Store-Apps verwendet wird), anstelle des traditionellen .xap-Dateiformats von Windows Phone. Anwendungen, die für WP8.1 gebaut werden, können semantische Zoom, sowie Zugriff auf Single Sign-on mit einem Microsoft-Konto. Das Windows Phone Jetzt auch Updates Apps automatisch speichern. Der Speicher kann manuell auf Updates für Anwendungen auf einem Gerät überprüft werden. Es fügt auch die Option, Anwendungen zu aktualisieren, wenn nur auf Wi-Fi. App-Entwickler können Apps mit Hilfe von C# / Visual Basic.NET .(NET,) C+ (CX) oder HTML5/JavaScript, wie für Windows 8.Developers werden auch in der Lage sein, "universal apps" für Windows Phone 8.1 und Windows 8 zu bauen, die fast alle Code teilen, außer für die spezifische auf der Plattform, wie Benutzeroberfläche und Telefon-APIs. Alle universellen Apps, die unter Windows 8.1 installiert wurden, werden automatisch im Abschnitt "Meine Apps" des Benutzers unter Windows Phone 8.1. angezeigt. Apps für Windows Phone 8 und Windows Phone 7 automatisch unter Windows Phone 8.1, aber Apps für Windows Phone 8.1 gebaut werden nicht auf einer vorherigen Version von Windows Phone laufen. Windows Phone Store Der Windows Phone Store wurde in Windows Phone 8.1 umgestaltet, um mehr Informationen zu erhalten. App-Sammlungen, die zuvor in einer anderen Seite sichtbar waren, sind nun vollständig auf der vorderen Spalte des Stores zu sehen. Es gibt auch keinen Unterschied mehr zwischen Games und anderen Apps; beide erscheinen jetzt in der App-Liste, obwohl Kategorien für Apps und Spiele (wie "die beliebtesten Spiele" oder "die beliebtesten Apps)" noch getrennt sind. App-Bewertungen wurden entwickelt, um diejenigen von Windows 8.1, mit horizontalen Bars hinzugefügt, um anzuzeigen, wie viele 5-Sterne-, 4-Sterne- oder 3-Sterne-Bewertungen eine App erhalten hat. App-Screens haben jetzt nicht mehr ihre eigene Seite, sondern können auf der Unterseite der App-Beschreibungsseite betrachtet werden. Darüber hinaus das Windows Phone Store enthält jetzt einen Abschnitt "Meine Apps" unter dem Dreipunkt-Menü, mit dem Benutzer jede zuvor gekaufte App neu installieren können. Windows Phone 8.1 Unterstützung endete am 11. Juli 2017. Der Windows Phone Store unter Windows Phone 8.1 wurde am 16. Dezember 2019 gesperrt. Neue und überarbeitete Apps Battery Saver fügt die Fähigkeit hinzu, die Batterienutzung zu verfolgen und Profile zu bestimmen, die den Stromverbrauch senken. Darüber hinaus wurde die "Background Tasks"-Seite, die es einem Benutzer ermöglicht, im Hintergrund eine einzelne App zu stoppen oder zuzulassen, vom Menü Einstellungen auf Battery Saver verschoben. Neben der Möglichkeit, eine Hintergrundaufgabe nur vom Laufen zu stoppen, können Benutzer nun Profile festlegen, die verhindern, dass bestimmte Apps nur laufen, wenn der Batteriestand unter einem bestimmten Prozentsatz liegt. Storage Sense ermöglicht es Benutzern, Dateien und Apps zwischen der Festplatte ihres Telefons und einem Mikro zu bewegen SD-Karte, und enthält Funktionen, die zuvor in der Einstellungs-Sektion, die Benutzer die Möglichkeit, temporäre Dateien zu löschen, um Speicher und Deinstallieren Anwendungen frei. Wi-Fi Sense markiert automatisch in Windows Phones, um vertrauenswürdige Wi-Fi Hotspots. Es hilft auch, Ihre eigenen Wi-Fi-Anmeldeinformationen mit Ihren Freunden und Kontakten zu teilen, aber ohne den Sicherheitskompromiss, d.h. es teilt nur die WiFi-Verbindung, ohne Ihren Freunden das Passwort zu sagen.Die Kalender-App hat nun eine Wochenansicht mit aktuellem Wetter, ähnlich den Funktionen im Microsoft Outlook-Kalender, die für Desktop-Benutzer zur Verfügung stehen. Der Kalender fügt weiterhin Unterstützung für die Synchronisation von Google Kalendern mit Ihrem Windows Phone hinzu. Karten wurden mit Unterstützung für die Luftsicht, 3D-Mapping und einen dynamischen Kompass überholt. Local Scout, die aufgrund der Implementierung von Cortana aus Windows Phone 8.1 in den Vereinigten Staaten entfernt wurde, wurde zu Maps verschoben. Die Karte zeigt auch nahe gelegene WiFi-Hotspots, falls vorhanden, in Ihrer Lage. Anruf und Skype Die Dialer-App fügt eine "Speed Dial"-Seite hinzu und Anrufe von einem einzigen Anrufer in Call History werden nun gruppiert. Durch Klicken auf die Gruppe werden einzelne Anrufdetails wie die Zeit und das Datum des Anrufs angezeigt. Neben jedem Anrufer wurde eine Schaltfläche hinzugefügt, mit der unidentifizierte Anrufer in die Kontaktliste eines Benutzers aufgenommen werden können oder Informationen über einen vorhandenen Kontakt aufrufen können. Benutzer können nun automatisch bestehende Telefonanrufe auf Skype-Videoanrufe innerhalb des Telefonanrufs UI aktualisieren, der auch mit größeren Tasten überarbeitet wurde. Zusätzlich zu einem großen Foto des Kontaktes erscheinen Text mit dem Namen des Benutzers und Telefonnummer jetzt an der Spitze des Bildschirms statt direkt über dem Dialer. Skype-Anrufe können auch direkt von Cortana initiiert werden. Multimedia Xbox Music und Xbox Video bieten Streaming-Services für Filme, Musik und TV-Shows und sind getrennt, im Gegensatz zu in früheren Versionen zusammengefügt. Sicher, Xbox Video hat jetzt integrierte Unterstützung für Videostreaming. Neben der Trennung seiner Musik- und Videostreaming-Dienste, fügt 8.1 auch Unterstützung für separate Lautstärkesteuerungen, Audio- und Videotranscodierung, Hardwarebeschleunigung, stereoskopische 3D und die Fähigkeit von Apps, Video unabhängig vom eingebauten Videorecorder des Betriebssystems zu erfassen und aufzunehmen. Darüber hinaus ist eine integrierte Unterstützung für das Streamen durch DLNA zu Monitoren und Fernsehbildschirmen, die von Microsoft als PlayTo bezeichnet werden, enthalten, sowie die Fähigkeit, Display von einem Telefon auf einen separaten Bildschirm zu spiegeln. Medienbearbeitungstools wurden ebenfalls verfeinert: Apps für langsame Videoaufnahme, Videoeffekte und Audioeffekte wurden hinzugefügt. Microsoft bietet derzeit zweiwöchige Updates, d.h. zweimal im Monat, zu beiden dieser Apps. Im August 2014 wurden auch A2DP und AVRCP unterstützt. Die Stock-Kamera-App wurde mit einem minimalistischen Design ähnlich wie die Kamera-App unter Windows 8.1. aktualisiert. Zusätzlich können Benutzer jetzt hochauflösende Fotos direkt auf OneDrive speichern, anstatt nur die Option, die 5MP-Version des Bildes in die Cloud hochzuladen. Multitasking Aufbauend auf Verbesserungen, die im dritten Update zu seinem Vorgänger gemacht wurden, fügt Windows Phone 8.1 Unterstützung für das Schließen von Apps hinzu, indem Sie auf sie in der Multitasking-Ansicht (aufgefordert durch eine Long-Drücke auf der Rückseite), die ähnlich ist, wie Multitasking unter Windows 8 und iOS arbeitet. Drücken der hinteren Taste stoppt nun eine App in der Multitasking-View statt zu schließen. Live-Fliesen Eine dritte Spalte von Live-Fliesen, die zuvor nur Windows-Handys mit 1080p und wählen Sie Telefone mit 720p-Bildschirmen, ist jetzt eine Option für alle Windows Phone 8.1 Geräte unabhängig von der Bildschirmgröße. Microsoft hat auch die Fähigkeit für Benutzer, lebende Fliesen mit einem Hintergrundbild zu Haut hinzugefügt. Mit der Aufnahme von Update 1 lässt WP8.1 Benutzer nun App-Fliesen übereinander ziehen, um Ordner von Apps auf dem Start-Bildschirm zu erstellen. Jede einzelne App innerhalb des Ordners kann noch als Live-Fliese erscheinen, und das Öffnen des Ordners erweitert es einfach auf dem Startbildschirm, so dass der Benutzer Apps umordnen und öffnen kann. Social Der Me-Hub in Windows Phone 8.1 wurde von einem einzigen Hub verwandelt, um alle Social Media-Konten zu einem einzigen Hub zu einem Betrachter zu aktualisieren und zu pflegen, der es den Nutzern ermöglicht, Nachrichten-Feeds aus sozialen Netzwerken anzuzeigen. Wenn Benutzer auf einen Facebook-Post klicken, zum Beispiel, werden sie sofort auf die Facebook-App umgeleitet, anstatt auf diesen Beitrag in der Me-Hub selbst, eine Funktion in früheren Versionen von Windows Phone zu mögen oder kommentieren. Das Benachrichtigungszentrum von Me-Hub für soziale Netzwerke wurde ebenfalls entfernt, da diese Funktion nun in das Action Center von Windows Phone 8.1 integriert ist. Zu den unterstützten sozialen Netzwerken im Me-Hub gehören Facebook, Foursquare, LinkedIn und Twitter, die auch vollständig in den Contacts-Hub integriert wurden."Die Leser, die es Nutzern ermöglichten, zwischen verschiedenen Chat-Diensten nahtlos umzuschalten, wurden ebenfalls aus der Messaging-App entfernt, die jetzt nur für Textnachrichten gilt.Andere Änderungen an der Messaging-App beinhalten die Möglichkeit, mehrere Textnachrichten zum Weiterleiten oder Löschen auszuwählen. Lockscreen Windows Phone 8.1 fügt die Fähigkeit für OEMs und einzelne Apps hinzu, ihre benutzerdefinierten Lock-Bildschirmthemen noch weiter anzupassen, indem die Schriftart und Orientierung von Zeit, Datum und Benachrichtigung Text. Benachrichtigungen und Einstellungen Ein neues Benachrichtigungszentrum namens "Action Center" wurde hinzugefügt und ermöglicht die Möglichkeit, einfache Einstellungen wie Volumensteuerungen zu ändern. Das Design des neuen Benachrichtigungsbereichs ermöglicht es dem Benutzer, z.B. drahtlose Netzwerke zu ändern, Bluetooth- und Flugzeugmodus ein- oder ausschalten zu lassen, und Zugriff auf "Driving Mode" von vier kundenspezifischen Boxen an der Oberseite des Bildschirms, während unter diesen vier horizontal platzierten Boxen neueste Textnachrichten und soziale Integration enthalten. Apps können auch standortspezifische Benachrichtigungen von Benutzern unter Hinzufügung einer neuen geofencing API senden. Keyboard Microsoft hat eine Word Flow-Tastatur in Windows Phone 8.1 hinzugefügt, die, ähnlich wie die Swype-Tastatur-Option auf Android-Geräten, ermöglicht Benutzern durch Buchstaben zu tippen. Wenn der Benutzer schwipiert, erzeugt die Tastatur automatisch Raum für das nächste Wort eingegeben werden. Die Tastatur wurde für seine Geschwindigkeit und Genauigkeit angepriesen und brachte Ruhm zu Microsofts Forschungsabteilung, als fünfzehnjähriger Lakeside School-Student Gaurav Sharma, mit einem Nokia Lumia 520 ausgestattet mit Windows Phone 8.1 und der "Word Flow"-Tastatur, brach die Guinness World Record für die weltweit schnellste Eingabe auf einem auf einem Mobiltelefon, das zuvor von einem Samsung Galaxy S4 Benutzer gehalten wurde, um acht Sekunden. Dieser Datensatz war kurzlebig, der später einen Monat später von Marcel Fernandes geschlagen wurde, der ein Viertel einer Sekunde schneller mit dem Fleksy Keyboard, einer konkurrierenden Tastatur auf iOS und Android fertigte. Da Flesky jedoch auf vorausschauende Textalgorithmen setzt, anstatt Gesten zu schweben, ist es fair zu sagen, dass "Word Flow" die weltweit am schnellsten schwebe Tastatur auf einem Mobiltelefon bleibt. Das Dateisystem Windows Phone 8.1 ermöglicht Apps, alle auf einem WP-Gerät gespeicherten Dateien anzuzeigen und den Inhalt des internen Dateisystems eines Geräts zu verschieben, zu kopieren, einzufügen und zu teilen. Durch diese Änderung wurden mehrere Datei-Explorer-Apps seit dem Debüt des Betriebssystems im Windows Phone Store veröffentlicht. Microsoft veröffentlicht seine eigene Datei Explorer App am 30. Mai 2014. Zusätzlich zu diesen Änderungen wurde SkyDrive nach Microsofts Beilegung eines Streits über das Sky-Markezeichen mit BSkyB komplett auf OneDrive über das Betriebssystem umbenannt. Benutzer werden auch mit mehreren Optionen präsentiert, wenn ein Windows Phone 8.1 Gerät über USB mit einem Computer verbunden ist. Windows Phone 8.1 unterstützt VPN und Bluetooth 4.0 LE. Mit der Veröffentlichung von Update 1 werden auch das Empfangen oder Senden von Daten durch VPN bei der Verbindung mit einem WLAN-Netzwerk und dem Standard Bluetooth PAN (Personal Area Network) 1.0 unterstützt. Apps Corner ist ein Kiosk-Modus, mit dem Benutzer die Nutzung auf eine ausgewählte Gruppe von Anwendungen beschränken und direkt in eine bestimmte App booten können.
