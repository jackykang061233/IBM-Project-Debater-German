Linode, LLC ist ein US-amerikanisches, privat geführtes Cloud-Hosting-Unternehmen, das virtuelle private Server bereitstellt. Es ist in Philadelphia, Pennsylvania. Geschichte Linode (ein Portmanteau der Wörter Linux und Knoten) wurde von Christopher Aker gegründet, die Mitte 2003 startete. Aker ist Absolvent der Full Sail University in Florida. Linode wechselte im März 2008 von UML auf Xen Virtualisierung und dann Mitte 2015 auf KVM. Er hat 2009 seinen Datensicherungsdienst gestartet. NodeBalancer, ein Last-Balancing-Service, gestartet im Juli 2011. Linode veröffentlichte Linode Managed, ein Vorfall-Response-Service für Unternehmen, im Jahr 2013. Linode eröffnete 2003 seine ersten Rechenzentren in Fremont, Kalifornien und Dallas. Es eröffnete anschließend Datenzentren in Atlanta im Jahr 2007; Newark, New Jersey, 2008; London im Jahr 2009; Tokio im Jahr 2011 und 2016;Frankfurt und Singapur im Jahr 2015; und Toronto, Mumbai und Sydney im Jahr 2019. Im Jahr 2015 kaufte Christopher Aker das historische Gebäude der Corn Exchange Bank an der North 3rd Street und der Arch Street in der Altstadt von Philadelphia. Nach Renovierungen verlegte Linode 2018 seinen Hauptsitz. Aker sagte, dass das Unternehmen Schwierigkeiten hatte, Arbeiter aus seinem früheren Vorort in Galloway Township, New Jersey anzuziehen, und hoffte, dass ein Philadelphia Standort würde helfen, Talent zu entwerfen. Produkte und Dienstleistungen Seine Flagship-Produkte sind Cloud-Hosting-Dienste mit mehreren Paketen zu verschiedenen Preispunkten. Ab November 2019 bietet Linode fünf Arten von Rechendiensten an, die auf spezifische Bedürfnisse der Kunden ausgerichtet sind, wie hohe Speicheranforderungen, dedizierte CPU oder GPU oder allgemeine Anwendungszwecke. Linode Block Speichern ermöglicht es Benutzern, ihre Serverspeicherkapazität mit Volumes auf Anfrage zu erweitern. Linode Backup ermöglicht es Kunden, ihre Server täglich, wöchentlich oder monatlich zu sichern. Linode Manager und NodeBalancer ermöglichen es Benutzern, mehrere Serverinstanzen über ein einziges System zu verwalten. Im September 2013 startete Linode Longview, ein Analysepaket für Linux. Linode hat am Ende Oktober 2019 seinen S3-Compatible Object Storage Service gestartet, um den Kunden große unstrukturierte Daten zu speichern. Dies war eine sehr gefragte Funktion der Kunden. Am 11. November 2019 auf der CNCF KubeCon kündigte Linode die Verfügbarkeit seines Managed Kubernetes Motordienstes an. Linode bietet mehrere Produkte und Dienstleistungen für seine Kunden. Compute Linode Kubernetes Engine (LKE) - Kubernetes Container als Service-Anwendung. GPU - On-Demand Cloud GPU (Graphical Processing Unit) berechnen Konfigurationen für parallele Bearbeitungs-Workloads wie maschinelles Lernen, wissenschaftliches Computing und Videoverarbeitung. Dedicated CPU - Dedicated CPU Instanzen laufen auf ihren eigenen CPU-Kernen in der Linode-Infrastruktur. Kein Austausch des Prozessors mit anderen Kunden. High Memory - High Memory-Compute-Konfigurationen, um speicherintensive Anwendungen auszuführen. Geteilte Linodes - Eine ausgewogene Berechnungskonfiguration für eine Vielzahl von Cloud-Anwendungen. Nanode - $5/Monats-Eintrags-Level-Compute-Paket, um eine voll funktionierende Linux-Distribution auszuführen. Storage Linode Object Storage - Hochverfügbare und skalierbare S3-Kompatible Objektspeicher, um unstrukturierte Daten zu speichern. Linode Block Storage - Blockieren Sie die Speicherung von Daten in Volumen oder Blöcken, die als einzelne Festplattenlaufwerke (HDD) wirken. Linode Backups - Automatischer Datensicherungsdienst. Networking VPC - Virtual Private Cloud zur Verwaltung des softwaredefinierten Netzwerks von Cloud-Ressourcen. VLAN - Virtual Local Area Network DDoS Protection - Erkennung und Minderung von verteilten Denial-of-Service (DDoS) Infrastrukturangriffen. NodeBalancers - Software-definierter, verwalteter Dienst zum Lastausgleich des Verkehrs. DNS Manager - Managed Domain Name System (DNS) Service auf Linode Infrastruktur. Next Generation Network - Linodes moderne weltweite Netzwerk-Infrastruktur, die globale Rechenzentren über ein privates Faser-Backbone verbindet. Management Tools Cloud Manager - Eine benutzerfreundliche Schnittstelle zur Bereitstellung und Verwaltung von virtuellen Linode-Maschinen, zur Konfiguration von Netzwerken und zur Steuerung von Benutzerkonten. Linode API - API (Application Programming Interface) für den programmatischen Zugriff auf die Linode-Plattform. Linode CLI - CLI (Command Line Interface) Kontozugriff von der Verwaltung von Linode Infrastruktur. Bilder - Erstellen Sie Kunden Linux Bilder, um Skalierung einfacher zu machen. StackScripts - Automatische Anpassung der Linode-Infrastruktur-Bereitstellungen mit StackScripts. Monitoring - Analysieren Sie die Infrastrukturleistung, die Nutzung von Track Transfers und erstellen Sie benutzerdefinierte Warnungen. Professional Services Managed - Ein Notfall-Responsor-Service, um Unternehmen zu helfen, teure Ausfallzeiten zu reduzieren.Professional Services - Cloud-Beratung, um Ihnen zu helfen, Ihre Dienstleistungen zu entwerfen, Standortmigrationen durchzuführen und Software bereitzustellen. Global Network Linode betreibt weltweit 11 Rechenzentren, die durch das Next Generation Network und umfangreiche Peering-Anordnungen des Unternehmens verbunden sind. IPv6 ist auf allen unterstützten Betriebssystemen von Linode vollständig aktiviert und kostenloser DDoS-Schutz ist auf allen virtuellen Maschinen des Unternehmens enthalten. Sicherheitsvorfälle Die Konten von acht Linode Kunden, die Bitcoin elektronische Währung gehalten wurden, wurden im März 2012 beeinträchtigt. Etwa 40.000 Bitcoins wurden gestohlen. Hack The Planet (HTP) Zugriff auf Linodes Webserver im Jahr 2013. Die Gruppe nutzte eine technische Sicherheitslücke im Anwendungsserver von Adobe ColdFusion. Linode sagte, dass HTP keine finanziellen sensiblen Informationen entschlüsseln konnte und alle Kontokennwörter zurücksetzen. Linode kündigte im Mai 2013 eine zweistufige Authentifizierung für seine Dienste an. Ab dem Weihnachtstag 2015 und bis zum 10. Januar wurde Linode von großen und häufigen DDoS-Angriffen getroffen, die durch einen "schlechten Schauspieler" verursacht wurden, der große Mengen an Botnet-Kapazität kaufte, um Linodes Geschäft erheblich zu beschädigen. Linode war das Opfer eines weiteren schweren DDoS-Angriffs über das Labor Day Wochenende 2016.
