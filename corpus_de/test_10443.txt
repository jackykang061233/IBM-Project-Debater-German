Formale Verfallbarkeit Eine formale Verfallbarkeit, Deduktivität, logischer Abfall oder Nichtsequitur (lateinisch für "it nicht") ist ein Fehler in der Struktur eines deduktiven Arguments, das das Argument ungültig macht. Man kann im Standardsystem der Logik klar zum Ausdruck kommen. Ein solches Argument wird immer als falsch angesehen. Das Vorhandensein der formalen Verfallbarkeit bedeutet nichts über die Räumlichkeiten des Arguments oder seinen Abschluss. Beide können tatsächlich wahr sein oder vielleicht sogar aufgrund des Arguments wahrscheinlicher sein; aber das deduktive Argument ist nach wie vor ungültig, weil der Abschluss nicht den beschriebenen Räumlichkeiten folgt. Abgesehen von der Erweiterung kann ein Argument eine formale Verfallbarkeit enthalten, auch wenn das Argument nicht ein deduktives Argument ist: z.B. ein induktives Argument, das die Grundsätze der Wahrscheinlichkeit oder der Kausalalität falsch anwendet, kann gesagt werden, dass es sich um einen formalen Fall handelt. "Seit deduktive Argumente hängen von den formalen Eigenschaften und induktiven Argumenten ab, die es nicht gibt, gelten formale Abzüge nur für deduktive Argumente." Eine logische Form wie "A und B" ist unabhängig von jeder besonderen Verbindung sinnvoller Vorwürfe. Logistische Form allein kann garantieren, dass in wahren Räumen ein echter Abschluss folgen muss. formale Logik macht jedoch keine solche Garantie, wenn eine Prämisse falsch ist; der Abschluss kann entweder wahr oder falsch sein. Jeglicher formaler Fehler oder logischer Abfall ist gleichfalls ungültig. Sowohl das Argument als auch alle seine Räumlichkeiten müssen für eine Erklärung gelten. Der Begriff der logischen Abneigung ist in einem Sinne, dass die Logik sich auf gültige Gründe bezieht, während ein Rückgang der Unsicherheit die Verwendung armer Gründe ist.
