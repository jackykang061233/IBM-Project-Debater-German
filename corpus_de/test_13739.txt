Eine Smartwatch ist ein tragbarer Computer in Form einer Uhr; moderne Smartwatches bieten eine lokale Touchscreen-Schnittstelle für den täglichen Gebrauch, während eine zugehörige Smartphone-App für Management und Telemetrie (wie Langzeit-Biomonitoring) sorgt. Während frühe Modelle grundlegende Aufgaben wie Berechnungen, digitale Zeitvergabe, Übersetzungen und Game-Playing ausführen könnten, haben Smartwatches 2010s allgemeinere Funktionalität näher an Smartphones, darunter mobile Apps, ein mobiles Betriebssystem und WiFi/Bluetooth-Konnektivität. Einige Smartwatches funktionieren als tragbare Medienplayer, mit FM-Radio und Wiedergabe von digitalen Audio- und Videodateien über ein Bluetooth-Headset. Einige Modelle, genannt Uhrentelefone (oder umgekehrt), haben mobile zelluläre Funktionalität wie Anrufe. Während interne Hardware variiert, haben die meisten eine elektronische visuelle Anzeige, entweder hinterleuchtete LCD oder OLED. Einige verwenden transflective oder elektronisches Papier, um weniger Strom zu verbrauchen. Sie werden in der Regel von einer wiederaufladbaren Lithium-Ionen-Batterie betrieben. Peripheriegeräte können digitale Kameras, Thermometer, Beschleunigungsmesser, Pedometer, Herzfrequenzwächter, Höhenmesser, Barometer, Kompasse, GPS-Empfänger, winzige Lautsprecher und microSD-Karten umfassen, die von vielen anderen Computern als Speichergeräte erkannt werden. Software kann digitale Karten, Scheduler und persönliche Organisatoren, Taschenrechner und verschiedene Arten von Uhrenflächen umfassen. Die Uhr kann mit externen Geräten wie Sensoren, drahtlosen Headsets oder einem Heads-up-Display kommunizieren. Wie andere Computer kann eine Smartwatch Informationen von internen oder externen Sensoren sammeln und Daten von anderen Instrumenten oder Computern steuern oder abrufen. Es kann drahtlose Technologien wie Bluetooth, WLAN und GPS unterstützen. Für viele Zwecke dient ein "watch computer" als Frontend für ein Remote-System wie ein Smartphone, das mit dem Smartphone über verschiedene drahtlose Technologien kommuniziert. Smartwatches werden vorangetrieben, insbesondere deren Design, Batteriekapazität und gesundheitsbezogene Anwendungen. Gesundheitsbezogene Anwendungen umfassen Anwendungen, die Herzfrequenz, SpO2, Workout etc. messen. Pulsar wurde ein Markenname, der später 1978 von Seiko erworben werden würde. 1982 wurde eine Pulsar-Uhr (NL C01) veröffentlicht, die 24 Ziffern speichern könnte, so dass es höchstwahrscheinlich die erste Uhr mit benutzerprogrammierbarem Speicher oder Memorybank-Uhr. XHTML-MP wird von XHTML Basic 1.0 abgeleitet, indem XHTML Module hinzugefügt werden, wobei spätere Versionen des Standards weitere Module hinzufügen. Für bestimmte Module hat XHTML-MP jedoch keine vollständige Implementierung, so dass ein XHTML-MP-Browser möglicherweise nicht vollständig auf alle Module passt. Der XHTML MP 1.2 DTD ist die aktuelle Empfehlung, die im März 2008 abgeschlossen wurde. XHTML Basic 1.1 wurde im Juli 2008 eine W3C-Empfehlung, die XHTML-MP 1.2 Dokumenttyp-Erklärung ersetzte Um als XHTML-MP zu validieren, muss ein Dokument eine ordnungsgemäße Dokumenttyp-Deklaration (DTD) oder DOCTYPE enthalten, je nach der Version der Spezifikation gefolgt Hinweis, dass eine Reihe von Revisionen ausgegeben wurden, um technische Fehler in den oben genannten DTDs zu korrigieren, und das DTD-Format ist komplexer und weniger weit als die von Standard-HTML unterstützt. MIME-Typen Der MIME-Typ für XHTML Mobile Profile ist Application/vnd.wap.xhtml+xml". Die Konformierung von Benutzern sollte auch Anwendung/xhtml+xml und Text/html akzeptieren. Viele Desktop-Browser validieren nur XHTML-MP zur Anzeigezeit, wenn ein XML MIME-Typ angegeben ist.
