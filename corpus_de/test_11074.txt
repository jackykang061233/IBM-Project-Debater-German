Windows Phone 8 ist die zweite Generation des Betriebssystems Windows Phone von Microsoft. Es wurde am 29.10.2012 veröffentlicht und wie sein Vorgänger eine flache Benutzeroberfläche auf der Basis der Metro-Designsprache darstellt. Windows Phone 8.1, das im April 2014 vorgestellt wurde, war es gelungen. Windows Phone 8 ersetzt die Windows CE-basierte Architektur in Windows Phone 7 mit dem Windows NT Kern, der in Windows 8 Windows Phone 7-Geräten gefunden wurde, kann Windows Phone 8 nicht ausführen oder aktualisieren und neue speziell für Windows Phone 8 erstellte Anwendungen werden nicht für Windows Phone 7 Geräte bereitgestellt. Entwickler können ihre Apps sowohl für Windows Phone 7 als auch für Windows Phone 8 Geräte bereitstellen, indem sie beide Plattformen über die richtigen Softwares in Visual Studio ausrichten. Windows Phone 8 Geräte werden von Microsoft Mobile (ehemals Nokia), HTC, Samsung und Huawei hergestellt. Geschichte Juni 20, 2012 stellte Microsoft Windows Phone 8 (coded Apollo) vor, eine dritte Generation des Windows-Telefon-Betriebssystems für die Freisetzung später 2012. Windows Phone 8 ersetzt seine vormalige Windows CE-basierte Architektur mit einer auf Windows NT Kern basierenden Architektur und teilt viele Komponenten mit Windows 8 und ermöglicht den Entwicklern leicht Hafenanwendungen zwischen den beiden Plattformen. Windows Phone 8 erlaubt auch Geräte mit größeren Bildschirmen (die vier bestätigten Größen sind "WVGA 800 x480 15:9","WXGA 1280×768 15:9",720p1280×720 16:9","1080p 1920x1080 16:9" Entschließungen) und Multi-core-Verarbeiter, NFC (die in erster Linie verwendet werden können, um Inhalte zu teilen und Zahlungen durchzuführen), Rückhaltbarkeit mit Windows Phone 7 Apps, verbesserte Unterstützung für abnehmbare Speicher (d. h. jetzt besser funktioniert diese Lagerung auf Windows und Android), eine Neugestaltung der Karten, die auf der Basis der neuen CD-Karten und auf den Bildschirmen basiert. Windows Phone 8 wird darüber hinaus weitere Merkmale auf dem Unternehmenmarkt umfassen, wie Gerätemanagement, BitLocker Verschlüsselung und die Fähigkeit, einen privaten Markt zu schaffen, um Apps an Mitarbeiter zu verteilen – Merkmale, die erwartet werden, dass sie die Geschäftsmöglichkeiten der vorherigen Windows-Plattform erfüllen oder übersteigen. Windows Phone 8 unterstützt darüber hinaus über-the-air-Aktualisierungen, und alle Windows Phone 8-Geräte erhalten Software-Unterstützung für mindestens 36 Monate nach ihrer Freisetzung. Windows Phone 8 wird nicht als Aktualisierung für bestehende Windows Phone 7 Geräte zur Verfügung gestellt. Microsoft veröffentlichte stattdessen Windows Phone 7.8 als Aktualisierung für Windows Phone 7 Geräte, die mehrere Merkmale wie den neugestalteten Heimbildschirm unterstützten.
