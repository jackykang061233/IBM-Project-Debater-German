HCL Connections ist eine Web 2.0 Enterprise Social Software-Anwendung, die ursprünglich von IBM entwickelt und im Juli 2019 von HCL Technologies erworben wurde. Verbindungen ist eine Enterprise-Kooperationsplattform, die Teams hilft, effizienter zu arbeiten. Verbindungen sind Teil der HCL-Kollaborations-Suite, die auch Notes / Domino, Gleichzeit, Portal und Verbindungen enthält. Übersicht Connections wurde auf der Lotusphere 2007 bekannt gegeben und Version 1.0 wurde am 27. Juni dieses Jahres ausgeliefert. HCL Connections hat folgende Komponenten: Homepage - ein Portal, das Informationen aus vielen Quellen föderieren kann. Microblogging - vor allem verwendet, um mit Updates aus dem gesamten sozialen Netzwerk über die Homepage zu bleiben Profile - ein soziales Netzwerk-Service - in erster Linie verwendet, um Menschen in der Organisation durch Know-how, aktuelle Projekte und Verantwortung zu finden. Startseite, Mikroblogs und Tags. Gemeinschaften - ein gemeinsamer Raum für die Zusammenarbeit mit einem Diskussionsforum. Ideation - Bietet die Fähigkeit, Crowdsource Ideen Media Gallery - Verwendet, um Fotos und Videos zu teilen und diese Videos vom Server Blogs zu streamen - ein Blogging-Service. Lesezeichen - ein Social Bookmarking Service. Aktivitäten - ein Task-Management-Tool für Gruppen von Personen, die an einem bestimmten Projekt oder einer Aufgabe zusammenarbeiten. Dateien - eine Inhaltsbibliothek zum Speichern, Teilen und Überarbeitung von Computerdateien. Wikis - ein Wiki-System für das Veröffentlichen und Bearbeiten von Inhalten. Foren - ein Internet-Forum-System, wo Menschen Fragen stellen können, ihre Erfahrungen teilen und Themen von gemeinsamem Interesse diskutieren. Suche - für die Suche über IBM Connections. Auf der Lotusphere 2010, IBM Vorschau-Funktionen für zukünftige Releases geplant, einschließlich Änderungen, wie nutzergenerierte Inhalte moderiert Compliance/Auditing-Funktionen zusätzliche Integration mit IBM WebSphere Portal und Microsoft SharePoint erweiterte mobile Unterstützung HCLConnections verwendet offene Standards, einschließlich ATOM und RSS, um mit anderen Anwendungen zu integrieren und bietet eine REST-ähnliche API für Entwickler. Widgets können in HCL-Verbindungen hinzugefügt werden, einschließlich der von Google Gadgets und anderen Dienstleistungen sowie maßgeschneiderte entwickelt. HCL Connections wurde als führendes Produkt im Social-Software-Markt von Unternehmen beschrieben und IDC nannte es 2011 den weltweiten Marktführer im Bereich Marketshare. Komponenten Die zehn HCL Connections-Komponenten werden nach dem serviceorientierten Architekturkonzept auf einer Reihe von Dienstleistungen aufgebaut. Diese Komponenten sind in Form von J2EE-Anwendungen, die auf IBM WebSphere Application Server gehostet werden. Durch diese Ausgestaltung können die Komponenten unabhängig voneinander gehostet werden und sehr große Bereitstellungen unterstützt werden. Homepage Die Homepage dient als Portal für die soziale Zusammenarbeit des Nutzers. Aus der Box HCL Connections bietet 20 Widgets, die vom Benutzer angepasst werden können. IT-Administratoren können neue Widgets und OpenSocial Gadgets für die Nutzung durch Benutzer hinzufügen. Das Homepage-Modul enthält auch ein 'recent updates'-Display, das Änderungen wie neue Content-Posts und Status-Updates zeigt, die für den Benutzer relevant sind. Es wird auch ein System zur Aktualisierung des Nutzerstatus über das Microblogging bereitgestellt. Profile Profile Profiles bietet ein Online-Verzeichnis von Personen innerhalb einer Organisation. Die Menschen können nach Kriterien wie Schlüsselwörter, Namen, Verantwortlichkeiten, Interessen, Projekte, die sie sind, Know-how, Geschäftsbeziehungen, Tags oder deren Standort Das Profil einer Person enthält in der Regel ihren Namen, Job-Rolle, Basis-Standort, Reporting-Kette und Details zu den IBM Connection Bookmarks, Aktivitäten, Gemeinschaften und Blogs, an denen sie teilnehmen. Profile können angepasst werden und neue Felder hinzugefügt Bookmarks Bookmarks Bookmarks ist ein Social Bookmarking-Service, der es den Menschen erlaubt, den Web-Content zu buchen, zu markieren und zu teilen. Bookmarks können über Stichwort, Tags und die Person, die das Lesezeichen erstellt haben, lokalisiert werden. Neben der einfachen Suche können Bookmarks die Suchergebnisse dynamisch verfeinern, indem der Nutzer die Nutzer oder andere interessierte Tags identifiziert. Bookmarks können Lesezeichen über Standard-Feeds ausgeben und eine API bereitstellen, damit Drittanbieter-Tools mit ihr integriert werden können. Ab Connections 2.0.1 kann die API für: Suche von Lesezeichen verwendet werden Erhalten Sie eine Liste von Lesezeichen-Tags Erhalten Sie eine Liste von beliebten Lesezeichen Hinzufügen von Lesezeichen zu einer Web-Seite Hinzufügen einer Liste von Lesezeichen zu einer Web-Seite Hinzufügen von beliebten Lesezeichen zu einer Web-Seite Erstellen von Lesezeichen Löschen Lesezeichen Lesezeichen abrufen Aktualisieren von Lesezeichen Aktivitäten Aktivitäten Aktivitäten ist ein Task-Management-System, das Gruppen von Menschen ermöglicht, einfach an einer Aufgabe zu arbeiten. Aktivitäten sind in einer geschachtelten Baumhierarchie strukturiert, wo Einträge, To-do-Elemente und Abschnitte von der Wurzelaktivität abzweigen. Jede Nicht-Sektionsentität in einer Aktivität kann Rich-Text und benutzerdefinierte Felder für Dateien, Links, Text, Personen und Daten enthalten. To-Do-Elemente können auch ein Datum haben und eine Person, der To-Do zugewiesen ist. WikisWikis sind eine neue Möglichkeit, die IBM Connections 2.5 hinzugefügt wird, um die kollaborative Erstellung von Webinhalten zu erleichtern. Wikis sind Sammlungen von Seiten zu einem bestimmten Thema. Wiki-Mitglieder können auf den Seiten bearbeiten oder kommentieren oder eigene Seiten hinzufügen. Teams können Wikis nutzen, um einen zentralen Ort zu schaffen, um an einem Projekt mitzuarbeiten. Dateien Dateien Dateien ist ein persönlicher Datei-Sharing-Service, der Teil von IBM Connections 2.5 ist, verwendet, um Dateien hochzuladen, zu teilen, Tag, empfehlen und kommentieren. Die Gemeinschaften ermöglichen eine ad hoc und geplante Zusammenarbeit um ein Projekt oder ein Interessengebiet. A Verbindungen Community kann eine eigene Mediengalerie, Veranstaltungskalender, Idee, Mikroblogging, Blog, Foren, Lesezeichen, Aktivitäten, Feeds, Mitgliederliste, Wiki und Dateien haben. Blogs Blogs bieten Blogging-Funktionalität für Gruppen und Einzelpersonen. Foren Forums wird für Diskussionen verwendet.
