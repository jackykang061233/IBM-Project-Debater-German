Ein Key-Generator (key-gen) ist ein Computerprogramm, das einen Produktlizenzschlüssel, wie eine Seriennummer, erzeugt, die für die Verwendung einer Software-Anwendung erforderlich ist. Keygens können von Softwareherstellern für Lizenzierungssoftware in kommerziellen Umgebungen, in denen Software für eine ganze Website oder ein Unternehmen lizenziert wurde, rechtmäßig verteilt werden, oder sie können unter Umständen von Urheberrechtsverletzung oder Softwarepiracy unrechtmäßig verteilt werden. Illegitimate-Key-Generatoren werden typischerweise von Software-Crackern in der Warez-Szene und demoscene verteilt. Diese Keygens spielen oft "Keygen-Musik", die die Genres Dubstep oder Chiptunes im Hintergrund enthalten und künstlerische Benutzeroberflächen haben kann. Softwarelizenzierung Eine Softwarelizenz ist ein Rechtsinstrument, das die Nutzung und Verteilung von Computersoftware regelt. Oft werden solche Lizenzen durch die Implementierung in der Software ein Produktaktivierungs- oder digitales Rechtemanagement (DRM)-Mechanismus durchgesetzt, um eine unbefugte Nutzung der Software zu verhindern, indem eine Codesequenz ausgegeben wird, die bei der Aufforderung oder Speicherung in ihrer Konfiguration in die Anwendung eingegeben werden muss. Schlüsselüberprüfung Viele Programme versuchen, Lizenzschlüssel über das Internet zu überprüfen oder zu validieren, indem eine Sitzung mit einer Lizenzierung Anwendung des Software-Verlags. Fortgeschrittene Keygens umgehen diesen Mechanismus und beinhalten zusätzliche Funktionen zur Schlüsselüberprüfung, beispielsweise durch Generieren der Validierungsdaten, die sonst von einem Aktivierungsserver zurückgegeben werden würden. Wenn die Software eine Telefonaktivierung bietet, könnte das Keygen den richtigen Aktivierungscode generieren, um die Aktivierung zu beenden. Eine weitere Methode, die verwendet wurde, ist die Aktivierungsserver-Emulation, die den Programmspeicher aufruft, um das Keygen als de facto Aktivierungsserver zu sehen. Multi-Keygen Ein Multi-keygen ist ein Keygen, das eine Schlüsselgeneration für mehrere Softwareanwendungen bietet. Multi-Keygene werden manchmal über einzelne Keygen freigegeben, wenn eine Reihe von Produkten den gleichen Algorithmus zur Generierung von Produktschlüsseln erfordert. In diesem Fall muss nur ein einziger innerhalb des Schlüssels kodierter Wert geändert werden, um ein anderes Produkt anzusprechen. Autoren und Vertrieb Unbefugte Keygens, die typischerweise gegen Software-Lizenzbedingungen verstoßen, werden von Programmierern geschrieben, die in der Reverse Engineering und Software-Cracking, oft Cracker genannt, um Kopierschutz von Software oder digitale Rechte-Management für Multimedia zu umgehen. Keygens können Sie auf Warez-Seiten oder über Peer-to-Peer (P2P)-Netzwerke herunterladen. Malware Keygens Viele unbefugte Keygens, die über P2P-Netzwerke oder anderweitig verfügbar sind, enthalten schädliche Nutzlasten. Diese Key-Generatoren können oder können keinen gültigen Schlüssel erzeugen, aber die eingebettete Malware, die gleichzeitig unsichtbar geladen wird, kann beispielsweise eine Version von CryptoLocker (ransomware) sein. Antivirus-Software kann Malware in Keygen eingebettet entdecken; solche Software identifiziert oft auch unbefugte Keygens, die keine Nutzlast als potenziell unerwünschte Software enthalten, oft Etikettierung sie mit einem Namen wie Win32/Keygen oder Win32/Gendows. Schlüssel Ein Schlüsselwechsler oder Keychan ist eine Variation eines Keygens. Ein Keychan ist ein kleines Stück Software, die den Lizenzschlüssel oder die Seriennummer eines bestimmten Stücks proprietärer Software auf einem Computer installiert ändert.
