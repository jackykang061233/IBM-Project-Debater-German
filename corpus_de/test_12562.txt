Ein Pocket PC (P/PC, PPC) ist eine Klasse von persönlichen digitalen Assistenten (PDA), die das Windows Mobile oder Windows Embedded Compact Betriebssystem betreibt, das einige der Fähigkeiten moderner Desktop-PCs hat. Der Name wurde von Microsoft im Jahr 2000 als Rebranding der Palm-Size-PC-Kategorie eingeführt. Einige dieser Geräte hatten auch integrierte Telefon- und Datenfunktionen, die als Pocket PC Phone Edition oder einfach als Smartphone bezeichnet wurden. Ab 2010 gab es Tausende von Anwendungen für Handhelds, die sich an die Microsoft Pocket PC-Spezifikation halten, von denen viele Freeware waren. Microsoft-konform Pocket-PCs können mit vielen Add-Ons wie GPS-Empfängern, Barcode-Lesern, RFID-Lesern und Kameras verwendet werden. 2007, mit dem Aufkommen von Windows Mobile 6.0, Microsoft fiel den Namen Pocket PC zugunsten eines neuen Namensschema: Windows Mobile Classic (früher Pocket PC:) Geräte ohne integriertes Telefon; Windows Mobile Professional (früher Pocket PC Phone Edition:) Geräte mit integriertem Telefon und Touchscreen; Windows Mobile Standard (früher Smartphone:) Geräte mit integriertem Telefon, aber ohne Touchscreen.
