Vakuumröhrendioden Eine Thermiondiode ist eine Thermion-Ventileinrichtung, bestehend aus einer geschlossenen, evakuierten Glas- oder Metallhülle, die zwei Elektroden enthält: eine Kathode und eine Platte. Die Kathode wird entweder indirekt beheizt oder direkt beheizt. Wird eine indirekte Heizung eingesetzt, ist eine Heizung in der Hülle enthalten. Im Betrieb wird die Kathode auf rote Wärme erwärmt, etwa 800–1.000 °C (1.470–1,830 °F). Eine direkt beheizte Kathode besteht aus Wolframdraht und wird durch einen Strom erhitzt, der von einer externen Spannungsquelle durchströmt wird. Eine indirekt beheizte Kathode wird durch Infrarotstrahlung von einem nahegelegenen Heizer, der aus Nichrome-Draht besteht, erhitzt und mit Strom versorgt, der von einer externen Spannungsquelle bereitgestellt wird. Diese haben eine geringe Arbeitsfunktion, was bedeutet, dass sie leichter Elektronen emittieren als die unbeschichtete Kathode. Die Platte, die nicht erwärmt wird, emittiert keine Elektronen, sondern kann sie absorbieren. Die zu rektifizierende Wechselspannung wird zwischen Kathode und Platte angelegt. Wenn die Plattenspannung gegenüber der Kathode positiv ist, zieht die Platte elektrostatisch die Elektronen von der Kathode an, so dass ein Strom von Elektronen durch das Rohr von Kathode zu Platte fließt. Wenn die Plattenspannung gegenüber der Kathode negativ ist, werden keine Elektronen von der Platte emittiert, so dass kein Strom von der Platte zur Kathode gelangen kann.
