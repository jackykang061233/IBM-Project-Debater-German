In der Politikwissenschaft ist eine Initiative (auch bekannt als Volksinitiative oder Bürgerinitiative) ein Mittel, mit dem eine Petition, die von einer bestimmten Anzahl von registrierten Wählern unterzeichnet wird, eine Regierung dazu zwingen kann, entweder ein Gesetz zu erlassen oder eine öffentliche Abstimmung in der Gesetzgebung in der sogenannten indirekten Initiative zu halten, oder in direkter Initiative, wo die Stellung in ein Plebiszit oder Referendum gesetzt wird, in dem, was als ein Volks initiiertes Referendum oder Bürgerin genannt wird. In einer indirekten Initiative wird zunächst eine Maßnahme auf den Gesetzgeber Bezug genommen und dann nur dann eine Volksabstimmung abgegeben, wenn sie nicht von der Gesetzgeberin erlassen wird. Wird das vorgeschlagene Gesetz von der Gesetzgebung abgelehnt, so kann die Regierung gezwungen werden, den Vorschlag einem Referendum zu unterbreiten. Die Initiative kann dann in Form einer direkten Initiative oder einer indirekten Initiative erfolgen. In einer direkten Initiative wird eine Maßnahme direkt an ein Referendum gerichtet. Die Abstimmung kann auf einer vorgeschlagenen Bundesebene, einer Satzung, einer Verfassungsänderung, einer Chartaänderung oder einer lokalen Verordnung erfolgen oder die Exekutive oder Rechtsvorschrift dazu verpflichten, das Thema zu betrachten, indem es der Tagesordnung unterstellt wird. Es ist eine Form der direkten Demokratie. Durch eine direkte Initiative wird eine Initiative direkt auf den Wahlgang für Wähler übertragen oder ablehnen. Die Maßnahme wird nicht zuerst der Gesetzgeber vorgelegt. Indirekte Initiative Eine indirekte Initiative wird von einem Gesetzgeber gewählt, nachdem aus der Abstimmungspopulation hinreichende Unterschriften gesammelt wurden. In den meisten Bereichen wird die Maßnahme nur dann einer späteren Volksabstimmung vorgelegt, wenn sie durch die Gesetzgebung geändert wird. Initiative zur Einstellung der Agenda Eine Initiative zur Festlegung der Tagesordnung ist eine Maßnahme, die durch Petition an einen Rechtsakt zur Prüfung vorgelegt wird. Der Gesetzgeber kann den Vorschlag ohne öffentliche Abstimmung billigen oder ablehnen. Diese Form oder Initiative ist häufiger als eine rechtsverbindliche direkte oder indirekte Initiative. Nach Land Überblick Amerika Brasilien In Brasilien erfordert eine Volksrechtsinitiative zwei Bedingungen, bevor sie an den Nationalkongress gesendet wird: Unterschriften von mindestens 1% der nationalen registrierten Wähler und mindestens 0,3% der Personen, die von jedem von mindestens fünf der 27 Bundesunitäten (die 26 Staaten plus das Bundesgebiet) wählen dürfen. Wenn beide Bedingungen erfüllt sind, ist der Kongress verpflichtet, über die Beteiligung der Initiative zu diskutieren und abzustimmen.
