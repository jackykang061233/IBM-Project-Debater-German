Ein Webbrowser (gemeinsam als Browser bezeichnet) ist die Anwendungssoftware für den Zugriff auf das World Wide Web. Wenn ein Benutzer eine Webseite von einer bestimmten Website anfordert, ruft der Webbrowser die notwendigen Inhalte von einem Webserver ab und zeigt dann die Seite auf dem Gerät des Nutzers an. Ein Web-Browser ist nicht dasselbe wie eine Suchmaschine, obwohl die beiden oft verwirrt sind. Eine Suchmaschine ist eine Website, die Links zu anderen Websites bietet. Um jedoch mit dem Server einer Website zu verbinden und ihre Webseiten anzuzeigen, muss ein Benutzer einen Webbrowser installiert haben. Web-Browser werden auf einer Reihe von Geräten verwendet, darunter Desktops, Laptops, Tablets und Smartphones. Im Jahr 2020 nutzten schätzungsweise 4,9 Milliarden Menschen einen Browser. Der am häufigsten verwendete Browser ist Google Chrome, mit einem 64% globalen Marktanteil auf allen Geräten, gefolgt von Safari mit 19 % Geschichte Der erste Webbrowser namens WorldWideWeb wurde 1990 von Sir Tim Berners-Lee erstellt. Er rekrutiert Nicola Pellow, um den Zeilenmodus Browser zu schreiben, der Webseiten auf dummen Endgeräten angezeigt; es wurde 1991 veröffentlicht. 1993 war ein Wahrzeichen Jahr mit der Veröffentlichung von Mosaic, als "der weltweit erste populäre Browser" ausgezeichnet. Mit seiner innovativen grafischen Schnittstelle ist das World Wide Web System einfach zu bedienen und damit dem Durchschnittsmenschen zugänglicher. Dies wiederum löste den Internet-Boom der 1990er Jahre aus, als das Web sehr schnell wuchs. Marc Andreessen, der Leiter des Mosaic-Teams, startete bald sein eigenes Unternehmen Netscape, das 1994 den Mosaic-beeinflussten Netscape Navigator veröffentlichte. Navigator wurde schnell zum beliebtesten Browser. Microsoft debütierte Internet Explorer 1995, was zu einem Browser-Krieg mit Netscape führte. Microsoft konnte aus zwei Gründen eine dominante Position gewinnen: es gebündelt Internet Explorer mit seinem beliebten Windows-Betriebssystem und tat so als Freeware ohne Einschränkungen der Nutzung. Der Marktanteil des Internet Explorer stieg schließlich 2002 um über 95%. 1998 startete Netscape die Mozilla Foundation, um einen neuen Browser mit dem Open Source Software-Modell zu erstellen. Diese Arbeit entwickelte sich in den Firefox-Browser, zuerst veröffentlicht von Mozilla im Jahr 2004. Firefox erreichte 2011 einen Marktanteil von 28%. Apple veröffentlichte 2003 seinen Safari-Browser. Es bleibt der dominante Browser auf Apple-Geräten, obwohl es anderswo nicht populär geworden. Google debütierte seinen Chrome-Browser im Jahr 2008, die stetig Marktanteil von Internet Explorer nahm und wurde der beliebteste Browser in 2012. Chrome ist seitdem dominant geblieben. 2011 wurde die HTTPS Everywhere-Erweiterung veröffentlicht und die NoScript-Erweiterung erhielt zahlreiche Auszeichnungen. Im selben Jahr startete Mozilla die stabile Version von Tor Firefox, um das dunkle Web zu navigieren. Microsoft veröffentlichte seinen Edge-Browser im Jahr 2015 als Teil der Windows 10 Release.( Internet Explorer wird immer noch auf älteren Versionen von Windows verwendet.) In Bezug auf Technologie haben Browser ihre HTML-, CSS-, JavaScript- und Multimedia-Funktionen seit den 1990er Jahren stark erweitert. Ein Grund war es, anspruchsvollere Websites, wie Web-Anwendungen zu ermöglichen. Ein weiterer Faktor ist die signifikante Zunahme der Breitband-Konnektivität, die es Menschen ermöglicht, auf datenintensive Webinhalte wie YouTube-Streaming zuzugreifen, was während der Ära der Dialy-up-Modems nicht möglich war. Funktion Der Zweck eines Webbrowsers ist es, Inhalte aus dem Web zu holen und auf dem Gerät des Nutzers anzuzeigen. Dieser Vorgang beginnt, wenn der Benutzer einen Uniform Resource Locator (URL) wie https://en.wikipedia.org in den Browser eingibt. Fast alle URLs im Web starten mit entweder http: oder https: Das bedeutet, dass der Browser sie mit dem Hypertext Transfer Protocol (HTTP) abruft. Im Falle von https: Die Kommunikation zwischen dem Browser und dem Webserver wird zum Zwecke der Sicherheit und des Datenschutzes verschlüsselt. Sobald eine Webseite abgerufen wurde, zeigt die Rendering-Engine des Browsers sie auf dem Gerät des Benutzers an. Dazu gehören Bild- und Videoformate, die vom Browser unterstützt werden. Webseiten enthalten meist Hyperlinks zu anderen Seiten und Ressourcen. Jeder Link enthält eine URL, und wenn es angeklickt oder angeklickt wird, navigiert der Browser auf die neue Ressource. So beginnt der Prozess, Inhalte an den Benutzer zu bringen. Die meisten Browser verwenden einen internen Cache von Webseiten-Ressourcen, um Ladezeiten für spätere Besuche auf der gleichen Seite zu verbessern. Der Cache kann viele Elemente speichern, wie große Bilder, so dass sie nicht wieder vom Server heruntergeladen werden müssen. Cached Items werden in der Regel nur solange gespeichert, wie der Webserver in seinen HTTP-Antwortnachrichten festlegt.Einstellungen Webbrowser können typischerweise mit einem integrierten Menü konfiguriert werden. Je nach Browser kann das Menü Einstellungen, Optionen oder Einstellungen genannt werden. Das Menü hat verschiedene Arten von Einstellungen. Zum Beispiel können Benutzer ihre Startseite und Standard-Suchmaschine ändern. Sie können auch Standard-Webseitenfarben und Schriftarten ändern. Auch verschiedene Netzwerk-Konnektivität und Datenschutzeinstellungen sind in der Regel verfügbar. Datenschutz Während des Surfens werden Cookies, die von verschiedenen Websites empfangen werden, vom Browser gespeichert. Einige von ihnen enthalten Anmeldeinformationen oder Website Präferenzen. Andere werden jedoch verwendet, um das Nutzerverhalten über lange Zeiträume zu verfolgen, so dass Browser typischerweise einen Abschnitt im Menü zum Löschen von Cookies bereitstellen. Die Verwaltung von Cookies erfordert in der Regel eine Browsererweiterung. Browser bieten auch in der Regel Menüpunkte zum Löschen von Browserverlauf, Cache-Einträgen und anderen möglicherweise sensiblen Daten. Ein alternativer Ansatz ist der private Browsermodus, in dem die vorgenannten Elemente nicht vom Browser gespeichert werden. Aber dies ist eine temporäre Option, nur bei Verwendung dieses speziellen Modus aktiviert. Eigenschaften Die beliebtesten Browser haben eine Reihe von Funktionen gemeinsam. Sie protokollieren automatisch den Browserverlauf oder können in einem nicht protokollierten Privatmodus verwendet werden. Sie ermöglichen es den Benutzern auch Lesezeichen festzulegen und den Browser mit Erweiterungen anzupassen. Einige bieten einen Sync-Service. Die meisten Browser haben diese Benutzeroberfläche (UI) Funktionen: Erlauben Sie dem Benutzer, mehrere Seiten gleichzeitig zu öffnen, entweder in verschiedenen Browserfenstern oder in verschiedenen Registerkarten desselben Fensters. Zurück und vorwärts Tasten, um wieder auf die vorherige Seite besucht oder vorwärts auf die nächste. Erfrischen oder Nachladen und eine Stopptaste, um die aktuelle Seite neu zu laden und zu löschen. ( In den meisten Browsern wird die Stop-Taste mit der Reload-Taste zusammengeführt.) Eine Home-Taste, um zur Homepage des Benutzers zurückzukehren. Eine Adressleiste zur Eingabe der URL einer Seite und Anzeige. Eine Suchleiste, um Begriffe in eine Suchmaschine einzugeben. (In einigen Browsern wird die Suchleiste mit der Adressleiste zusammengeführt.) Die Einschränkungen von Touchscreens erfordern jedoch, dass mobile Browser eine einfachere UI als Desktop-Versionen haben. Der Unterschied ist für Benutzer, die an Tastaturkürzel gewöhnt sind, von Bedeutung. Sicherheit Web-Browser sind beliebte Ziele für Hacker, die Sicherheitslöcher nutzen, um Informationen zu stehlen, Dateien zu zerstören und andere bösartige Aktivitäten. Browser-Anbieter patchieren diese Sicherheitslöcher, so dass Benutzer stark ermutigt, ihre Browser-Software aktualisiert zu halten. Andere Schutzmaßnahmen sind Antivirensoftware und Vermeidung von bekannten bösartigen Websites.
