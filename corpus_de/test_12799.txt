Die Dynamic Host Configuration Protocol Version 6 (DHCPv6) ist ein Netzwerkprotokoll zur Konfiguration von Internet Protocol Version 6 (IPv6) Hosts mit IP-Adressen, IP-Präfixen und anderen Konfigurationsdaten, die für den Betrieb in einem IPv6-Netzwerk erforderlich sind. Es ist das IPv6-Äquivalent des Dynamic Host Configuration Protocol for IPv4. IPv6-Hosts können IP-Adressen automatisch intern unter Verwendung von stateless Address Autoconfiguration (SLAAC) generieren oder sie können Konfigurationsdaten mit DHCPv6 zugewiesen werden. IPv6-Hosts, die eine staatenlose Autokonfiguration verwenden, können andere Informationen als eine IP-Adresse oder Route benötigen. DHCPv6 kann verwendet werden, um diese Informationen zu erwerben, obwohl es nicht verwendet wird, um IP-Adressen zu konfigurieren. DHCPv6 ist nicht erforderlich, um Hosts mit den Adressen von Domain Name System (DNS) Servern zu konfigurieren, da diese mittels Neighbor Discovery Protocol konfiguriert werden können, was auch der Mechanismus für eine staatslose Autokonfiguration ist. Viele IPv6-Router, wie Router für Wohnnetze, müssen automatisch ohne Bedienereingriff konfiguriert werden. Solche Router benötigen nicht nur eine IPv6-Adresse für die Kommunikation mit vorgeschalteten Routern, sondern auch ein IPv6-Präfix zur Verwendung bei der Konfiguration von Geräten auf der stromabwärtigen Seite des Routers. Die DHCPv6 Prefix-Delegation bietet einen Mechanismus zur Konfiguration solcher Router. Operation Port Nummern DHCPv6 verwendet UDP Port Nummer 546 für Clients und Port Nummer 547 für Server. Identifiers DHCP eindeutige Kennung Die DHCP-Einzelkennung (DUID) wird von einem Client verwendet, um eine IP-Adresse von einem DHCPv6-Server zu erhalten. Es hat ein 2-byte DUID-Typfeld und ein Variable-Länge-Identifier-Feld bis zu 128 Bytes. Seine tatsächliche Länge hängt von seiner Art ab. Der Server vergleicht die DUID mit seiner Datenbank und liefert Konfigurationsdaten (Adresse, Leasingzeiten, DNS-Server usw.) an den Client. Die ersten 16 Bit eines DUID enthalten den DUID-Typ, von dem es vier Typen gibt. Die Bedeutung des verbleibenden DUID hängt vom Typ ab.
