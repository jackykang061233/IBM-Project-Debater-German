Windows Embedded Compact, früher Windows Embedded CE, Windows Powered und Windows CE, ist ein Betriebssystem-Unterfamilie von Microsoft im Rahmen seiner Windows Embedded-Produktfamilie entwickelt. Im Gegensatz zu Windows Embedded Standard, der auf Windows NT basiert, verwendet Windows Embedded Compact einen anderen Hybrid-Kernel. Microsoft lizenziert es an originale Gerätehersteller (OEMs), die ihre eigenen Benutzeroberflächen und Erfahrungen modifizieren und erstellen können, wobei Windows Embedded Compact die technische Grundlage dafür bietet. Die aktuelle Version von Windows Embedded Compact unterstützt x86 und ARM Prozessoren mit Board Support Paket (BSP) direkt. Windows CE war ein modulares/Komponenten-Betriebssystem, das als Grundlage für mehrere Geräteklassen wie Handheld PC, Pocket PC, Auto PC, Windows Mobile, Windows Phone 7 und mehr diente. Eigenschaften von Windows CE ist für Geräte mit minimalem Speicher optimiert; ein Windows CE-Kernel kann mit einem Megabyte Speicher laufen. Geräte sind oft ohne Festplattenspeicher konfiguriert und können als geschlossenes System ausgebildet sein, das keine Endbenutzererweiterung erlaubt (z.B. in ROM verbrennbar). Windows CE entspricht der Definition eines Echtzeit-Betriebssystems mit einer deterministischen Interrupt-Latenz.
