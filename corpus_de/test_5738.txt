Beim Computing ist eine Dateikürzung ein Griff in einer Benutzeroberfläche, die es dem Benutzer ermöglicht, eine Datei oder Ressource in einem anderen Verzeichnis oder Ordner von dem Ort zu finden, an dem sich die Verknüpfung befindet. In ähnlicher Weise ermöglicht eine Internet-Kürzung dem Benutzer eine Seite, Datei oder Ressource auf einer entfernten Internet-Site oder Website zu öffnen. Shortcuts werden typischerweise als eine kleine Datei implementiert, die ein Ziel URI oder GUID an ein Objekt enthält, oder den Namen einer Zielprogrammdatei, die die Verknüpfung darstellt. Die Verknüpfung kann zusätzlich Parameter angeben, die beim Start an das Zielprogramm übergeben werden sollen. Jeder Shortcut kann sein eigenes Icon haben. Shortcuts werden sehr häufig auf einem Desktop platziert, in einem Application Launcher Panel wie das Microsoft Windows Start Menü oder im Hauptmenü einer Desktop-Umgebung.
