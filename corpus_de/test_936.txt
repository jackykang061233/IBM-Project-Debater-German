Eine Satellitennavigation oder Satnav-System ist ein System, das Satelliten verwendet, um eine autonome geo-räumliche Positionierung zu ermöglichen. Es ermöglicht kleinen elektronischen Empfängern, ihre Lage (Länge, Breite und Höhe/Elevation) zu hoher Präzision (innerhalb einiger Zentimeter bis Meter) anhand von Zeitsignalen zu bestimmen, die entlang einer Sichtlinie durch Funk von Satelliten übertragen werden. Das System kann zur Bereitstellung von Position, Navigation oder zur Verfolgung der Position von etwas verwendet werden, das mit einem Empfänger (Satelliten-Tracking) ausgestattet ist. Die Signale erlauben es dem elektronischen Empfänger auch, die aktuelle Ortszeit auf eine hohe Präzision zu berechnen, was eine Zeitsynchronisation ermöglicht. Diese Verwendungen werden als Positioning, Navigation und Timing (PNT) bezeichnet. Satnav-Systeme arbeiten unabhängig von jeder Telefon- oder Internet-Empfang, obwohl diese Technologien die Nützlichkeit der erzeugten Positionierungsinformationen verbessern können. Ein Satellitennavigationssystem mit globaler Erfassung kann als globales Navigationssatellitensystem (GNSS) bezeichnet werden. Seit September 2020 sind das Global Positioning System (GPS,) Russlands Global Navigation Satellite System (GLONASS), Chinas BeiDou Navigation Satellite System (BDS) und das Galileo der Europäischen Union voll funktionstüchtig. Das japanische Quasi-Zenith Satellite System (QZSS) ist ein (US) GPS Satelliten-Augmentation System, um die Genauigkeit von GPS zu verbessern, mit Satellitennavigation unabhängig von GPS für 2023 geplant. Das Indian Regional Navigation Satellite System (IRNSS) plant, langfristig auf eine globale Version zu erweitern. Die globale Abdeckung für jedes System wird in der Regel durch eine Satellitenkonstellation von 18–30 mittelgroßen Erdorbit-Satelliten (MEO) erreicht, die sich zwischen mehreren Orbitalebenen verteilen. Die tatsächlichen Systeme variieren, verwenden aber Orbitalneigungen von >50° und Umlaufzeiten von etwa zwölf Stunden (auf einer Höhe von etwa 20.000 Kilometern oder 12.000 Meilen). Satellitennavigation ermöglicht Präzision bei der Lieferung von Waffen zu Zielen und erhöht ihre Lethalität und reduziert unbeabsichtigte Opfer von irregeleiteten Waffen.(Siehe geführte Bombe).
