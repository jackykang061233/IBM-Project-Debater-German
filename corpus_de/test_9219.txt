Eine Tapete oder Hintergrund (auch als Desktop-Tapete, Desktop-Hintergrund, Desktop-Bild oder Desktop-Bild auf Computern bekannt) ist ein digitales Bild (Foto, Zeichnung etc.) als dekorative Hintergrund einer grafischen Benutzeroberfläche auf dem Bildschirm eines Computers, Smartphones oder anderen elektronischen Geräts verwendet. Auf einem Computer werden Tapeten in der Regel auf dem Desktop verwendet, während auf einem Mobiltelefon dienen sie als Hintergrund für den Heimbildschirm. Obwohl die meisten Geräte mit einem Standard-Hintergrundbild kommen, ermöglichen moderne Geräte in der Regel Benutzer manuell das Hintergrundbild zu ändern. Der Begriff Wallpaper wurde in Microsoft Windows vor Windows XP verwendet (wo es den "Desktop Hintergrund" genannt wird), während macOS es als "Desktop-Bild" bezeichnet. Auf älteren Systemen, die kleine wiederholte Muster als Hintergrundbilder eingestellt werden konnten, wurde der Begriff Desktop-Muster verwendet. Geschichte Das X Window System war eines der frühesten Systeme, um Unterstützung für ein willkürliches Bild als Tapete über das xsetroot-Programm, die zumindest so früh wie die X10R3 Veröffentlichung im Jahr 1985 könnte den Bildschirm mit einer beliebigen festen Farbe oder einer beliebigen binären Bild X BitMap-Datei. 1989 wurde ein kostenloses Software-Programm namens xgifroot veröffentlicht, das es erlaubte, ein beliebiges GIF-Bild als Tapete zu verwenden, und im selben Jahr wurde das kostenlose xloadimage-Programm veröffentlicht, das eine Vielzahl von Bildformaten (einschließlich Farbbilder im Sun Rasterfile-Format) als Desktop-Hintergrund anzeigen könnte. Anschließend wurden eine Reihe von Programmen veröffentlicht, die Tapeten-Unterstützung für zusätzliche Bildformate und andere Features, wie das xpmroot-Programm (veröffentlicht 1993 als Teil von fvwm) und die xv-Software (veröffentlicht 1994). Das ursprüngliche Macintosh-Betriebssystem erlaubte nur eine Auswahl von 8×8-Pixel-Binärbild-Kachelmustern; die Fähigkeit, kleine Farbmuster zu verwenden, wurde 1987 in System 5 hinzugefügt. Mac OS 8 im Jahr 1997 war die erste Macintosh-Version, die integrierte Unterstützung für die Verwendung willkürlicher Bilder als Desktop-Bilder, anstatt kleine Wiederholungsmuster beinhaltete. Windows 3.0 im Jahr 1990 war die erste Version von Microsoft Windows mit Unterstützung für Tapete Anpassung kommen, und verwendet den Begriff Wallpaper für diese Funktion. Obwohl Windows 3.0 nur mit 7 kleinen Mustern (2 schwarz-weiß und 5 16-farbig) kam, konnte der Benutzer andere Bilder im BMP-Dateiformat mit bis zu 8-Bit-Farbe liefern (obwohl das System theoretisch in der Lage war, 24-Bit-Farbbilder zu handhaben, es tat dies durch dithering sie zu einer 8-Bit-Palette) ähnliche Tapeten-Features sonst fehlte in diesen Systemen. Ein Tapeten-Feature wurde 1991 in einer Beta-Version von OS/2 2.0 hinzugefügt. Aufgrund der weit verbreiteten Nutzung von Personal Computern, einige Tapeten sind enorm erkennbar und erhalten ikonischen kulturellen Status. Animierte Hintergründe Animierte Hintergründe (manchmal als Live-Hintergründe oder dynamische Hintergründe bezeichnet) bezieht sich auf Tapeten, die ein bewegtes Bild oder eine 2D / 3D-Szene als Betriebssystem-Hintergrund anstelle eines statischen Bildes aufweisen, es kann sich auch auf Tapeten beziehen, die in einer Playlist zykliert werden, oft mit bestimmten Übergangseffekten. Einige Betriebssysteme, wie das Android-Betriebssystem, bieten native Unterstützung für animierte Tapeten. Microsoft Windows Modern Windows-Systeme können konfiguriert werden, um durch Bilder aus einem Ordner in regelmäßigen Abständen zu zyklieren. Windows unterstützt nicht native animierte Hintergründe, aber Drittanbieter-Software kann installiert werden, um volle Unterstützung für die Platzierung von animierten Bildern, Videodateien, 2D- oder 3D-Szenen und Webseiten als Wallpaper. Ähnliche Funktionalitäten finden Sie in der Active Desktop-Funktion von Windows 98 und späteren Versionen. Google Android Live Wallpaper wurden in Android Eclair eingeführt, um native Unterstützung für animierte Tapeten zu bieten. Aus technischer Sicht sind Live-Tapeten Software-Anwendungen, die ein bewegtes Hintergrundbild bieten und die Benutzerinteraktion ermöglichen oder andere Hardware- und Software-Funktionen innerhalb des Gerätes nutzen können (Beschleuniger, GPS, Netzwerkzugriff, etc.) Apple macOS macOS unterstützt über das Desktop & Screen Saver Panel in seinen Systemeinstellungen, zum Radfahren durch eine Ordnersammlung von Bildern auf einem Zeitintervall oder beim Einloggen oder Aufwachen aus dem Schlaf. Seit macOS Mojave, der Benutzer kann auch einen "Dynamic Desktop" auswählen, der automatisch aktualisiert, um die Zeit des Tages visuell anzupassen. Darüber hinaus hat macOS die native Fähigkeit, einen Bildschirmschoner auf dem Desktop auszuführen; in dieser Konfiguration erscheint der Bildschirmschoner unter den Desktop-Symbolen anstelle der Systemtapete. macOS verfügt jedoch nicht über eine eingebaute Schnittstelle, sondern muss über Terminal-Befehle oder verschiedene Drittanbieter-Anwendungen erfolgen. iOS Dynamisch animierte Hintergründe wurden auch in iOS 7 und späteren Versionen eingeführt, jedoch sind sie auf die von Apple bereitgestellten beschränkt.
