Mastodon ist kostenlose und Open-Source-Software für die Durchführung von selbstgehosteten sozialen Netzwerkdienste. Es hat Mikroblogging-Funktionen ähnlich wie der Twitter-Service, die von einer großen Anzahl von unabhängig laufen Mastodon-Knoten (bekannt als Instanzen,) jeder mit seinem eigenen Code von Verhalten, Bedingungen von Service, Datenschutz-Optionen und Moderation Richtlinien angeboten werden. Jeder Benutzer ist Mitglied einer bestimmten Mastodon-Instanz, die als gefesseltes soziales Netzwerk interoperieren kann, so dass Benutzer auf verschiedenen Knoten miteinander interagieren können. Dies soll den Benutzern die Flexibilität geben, einen Server auszuwählen, dessen Richtlinien sie bevorzugen, aber den Zugang zu einem größeren sozialen Netzwerk behalten. Mastodon ist auch Teil des Fediverse-Ensembles von Serverplattformen, die gemeinsame Protokolle verwenden, mit denen Nutzer auch auf anderen kompatiblen Plattformen wie PeerTube und Friendica interagieren können. Das Mastodon Maskottchen ist ein Tier mit einem Stamm, ähnlich einem Mastodon oder Mammut, manchmal mit einem Tablet oder Smartphone dargestellt. Mit der Software veröffentlichte Nachrichten werden als Toots bezeichnet. Seit Version 2.9.0 bietet Mastodon standardmäßig einen einzigen Spaltenmodus für neue Benutzer. Im fortgeschrittenen Modus nähert sich Mastodon dem Mikroblogging-Benutzererlebnis von TweetDeck. Benutzer post Kurzform-Status-Nachrichten für andere zu sehen. Auf einer Standard-Mamodon-Instanz können diese Nachrichten bis zu 500 textbasierte Zeichen umfassen, eine Erweiterung der 280 Zeichengrenze von Twitter, obwohl zahlreiche Mastodon-Server den Quellcode aufgerufen haben, um eine größere Zeichengrenze zu ermöglichen. Posts werden Toots anstelle von Tweets aufgerufen, wie es auf Twitter der Fall ist. Benutzer schließen sich einem bestimmten Mastodon-Server an, anstatt einer einzigen Website oder Anwendung. Die Server sind als Knoten in einem Netzwerk verbunden, und jeder Server kann eigene Regeln, Kontoprivilegien verwalten und Nachrichten an und von anderen Servern teilen. Viele Server haben ein Thema basierend auf einem bestimmten Interesse. Es ist auch üblich, dass Server auf einer bestimmten Lokalität, Region oder Land basieren. Mastodon enthält eine Reihe von spezifischen Datenschutzfunktionen. Jede Nachricht hat eine Vielzahl von Datenschutzoptionen zur Verfügung, und Benutzer können wählen, ob die Nachricht öffentlich oder privat ist. Öffentliche Nachrichten, die auf einem globalen Feed angezeigt werden, bekannt als eine Timeline, und private Nachrichten werden nur auf den Zeitlinien der Anhänger des Benutzers geteilt. Nachrichten können auch aus Zeitlinien als nicht aufgeführt oder direkt zwischen den Benutzern markiert werden. Benutzer können auch ihre Konten als vollständig privat markieren. In der Zeitleiste können Meldungen mit einer optionalen "Content-Warnung"-Funktion angezeigt werden, die es den Lesern erfordert, auf den Inhalt zu klicken, um den Rest der Nachricht zu enthüllen. Mastodon-Server haben diese Funktion verwendet, um Spoiler zu verstecken, Warnungen auszulösen und nicht sicher für die Arbeit (NSFW) Inhalte, obwohl einige Konten verwenden die Funktion, um Links zu verstecken und Gedanken, die andere möglicherweise nicht lesen wollen. Mastodon aggregiert Nachrichten in lokalen und föderierten Zeitlinien in Echtzeit. Die lokale Timeline zeigt Nachrichten von Benutzern auf einem einzigartigen Server, während die föderierte Timeline Nachrichten auf allen teilnehmenden Mastodon Servern zeigt. Benutzer können über angeschlossene Mastodon-Server mit Benutzernamen ähnlich im Format zu vollständigen E-Mail-Adressen kommunizieren. Anfang 2017 zeichneten Journalisten wie Sarah Jeong Mastodon von Twitter für seinen Ansatz zur Bekämpfung von Belästigung aus, einem der größten Probleme von Twitter. Mastodon verwendet gemeinschaftsbasierte Moderation, in der jeder Server unerwünschte Inhaltsarten begrenzen oder filtern kann. Zum Beispiel, mastodon.social und mehrere andere Server verbieten Inhalte, die in Deutschland oder Frankreich illegal sind, einschließlich Nazi-Symbolismus, Holocaustleugnung und Diskriminierung. Server können auch wählen, um Nachrichten mit disparaging Inhalt zu begrenzen oder auszufiltern. Mastodons Gründer Eugen Rochko glaubt, dass kleine, eng miteinander verbundene Gemeinden mit unerwünschtem Verhalten effektiver umgehen als das kleine Sicherheitsteam eines großen Unternehmens. Technology Mastodon ist als kostenlose, webbasierte Software für föderierte Mikroblogging geschrieben, zu der jeder Code beitragen kann, und die jeder auf seiner eigenen Server-Infrastruktur laufen kann, wenn er wünscht, oder sich Servern anschließen kann, die von anderen Personen innerhalb des gefütterten Netzwerks betrieben werden.
