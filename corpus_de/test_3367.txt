Keras fungiert als Schnittstelle für die TensorFlow-Bibliothek. Bis zur Version 2.3, Keras unterstützte mehrere Backends, darunter TensorFlow, Microsoft Cognitive Toolkit, Theano und PlaidML. Ab Version 2.4 wird nur TensorFlow unterstützt. Entwickelt, um schnelle Experimente mit tiefen neuronalen Netzwerken zu ermöglichen, konzentriert es sich auf benutzerfreundlich, modular und erweiterbar. Es wurde im Rahmen der Forschungsarbeit des Projekts ONEIROS (Open-ended Neuro-Electronic Intelligent Robot Operating System) entwickelt, und sein Hauptautor und Betreuer ist François Chollet, ein Google-Ingenieur. Chollet ist auch der Autor des XCeption tiefen neuronalen Netzwerkmodells. Eigenschaften Keras enthält zahlreiche Implementierungen von häufig verwendeten neuralen Netzwerk-Bausteinen wie Schichten, Ziele, Aktivierungsfunktionen, Optimatoren und eine Vielzahl von Werkzeugen, um die Arbeit mit Bild- und Textdaten zu erleichtern, um die für das Schreiben von tiefen neuronalen Netzwerk-Code erforderliche Codierung zu vereinfachen. Der Code wird auf GitHub gehostet, und Community-Support-Foren umfassen die GitHub-Ausgaben-Seite, und einen Slack-Kanal. Neben Standard-Neuralnetzwerken unterstützt Keras konvolutionale und wiederkehrende neuronale Netzwerke. Es unterstützt andere häufige Gebrauchsschichten wie Dropout, Batch-Normalisierung und Pooling. Keras ermöglicht es Benutzern, tiefe Modelle auf Smartphones (iOS und Android), im Web oder auf der Java Virtual Machine zu produzieren. Es ermöglicht auch die Verwendung von verteilten Schulungen von Deep-learning-Modellen auf Clustern von Graphics Processing Units (GPU) und Tensor Processing Units (TPU).
